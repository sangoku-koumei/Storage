# 00_LCIQ全体設計書 —— 恋愛知性（Love Communication Intelligence Quotient）の全貌

## 1. プロジェクト概要
LCIQ（恋愛偏差値）は、恋愛・結婚・対人関係における「愛し、愛される力」を科学的に分析し、向上させるための教育プログラムです。

### 1.1 ビジョン
「愛の技術を義務教育へ」
感情に振り回されるのではなく、知性を持って運命を操り、持続可能な幸せを自ら創造できる個人を育成する。

### 1.2 ターゲットペルソナ
- **メイン**: 婚活に苦戦している、またはパートナーシップに不安を抱える30代〜40代の男女。
- **サブ**: 社内コミュニケーションを改善したい経営者、少子化対策に取り組む自治体担当者。

## 2. コア・コンセプト
恋愛は「運」でも「才能」でもなく、「後天的に習得可能な技術」である。

### 2.1 6つの構成要素
1. **認識力**: 自己の感情とバグの特定
2. **楽転力**: ポジティブなリフレーミング
3. **表現力**: 感情の言語化とコントロール
4. **共感力**: 他者理解とミラーニューロンの活用
5. **魅了力**: 自己信頼とエネルギーレベルの向上
6. **継続力**: 長期的なコミットメントと関係維持

## 3. カリキュラム構造
1. **診断フェーズ**: LCIQ診断による現状把握とボトルネックの特定。
2. **理論フェーズ**: 脳科学（扁桃体、オキシトシン）と心理学（愛着スタイル）の学習。
3. **実践トレーニングフェーズ**: ワークシートを用いたOSの書き換え。
4. **定着フェーズ**: 日々の習慣化とコミュニティでの実践。

## 4. 期待される成果
- **個人**: 自己肯定感の向上、理想のパートナーとの成婚、安定した家庭生活。
- **組織**: 心理的安全性の向上、従業員のメンタル安定、離職率の低下。
- **社会**: 出生率の向上、良好な人間関係の構築による社会全体の幸福度向上。
