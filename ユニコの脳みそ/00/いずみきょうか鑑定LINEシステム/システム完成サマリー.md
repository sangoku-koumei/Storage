```

   ↓（自動）

5. 受付メール送信
6. 翌日AI鑑定生成
7. 鑑定結果メール送信
```

### 有料鑑定

```
1. リッチメニュー「有料鑑定」タップ
2. PayPal.Me で決済（¥5,000 or ¥10,000）
   ↓（運営が確認）
3. manual_payments シートに1行追加
   ↓（1分以内に自動）
4. users/payments/applications シートに記録
5. フォームトークン生成
6. フォームURL案内メール送信
   ↓（ユーザー）
7. ユーザーがフォーム記入（相談内容）
8. フォーム送信（1回のみ）
   ↓（自動）
9. 翌日AI鑑定生成（3,500文字）
10. 鑑定結果メール送信
```

### 無料相談

```
1. リッチメニュー「無料相談」タップ

   ↓（自動）

4. 運営にメール通知
   ↓（運営が手動）
5. Zoom設定・日時決定
6. consult_decisions シートに記入
   ↓（1分以内に自動）
7. appointments シートに予約作成
```
