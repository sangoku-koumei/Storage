===============================================================================
【使い方】
===============================================================================

1. スプレッドシートで対象シートを開く
2. 該当するセクションをコピー（ヘッダー行から）
3. A1セルを選択
4. Ctrl+V（Windows）または Cmd+V（Mac）で貼り付け
5. データがタブ区切りで自動的に列に分かれます

- フォームURL案内メール送信
- processed が TRUE に更新

===============================================================================
重要な設定項目の変更
===============================================================================

【必ず変更する項目】

1. config シート：
   - openai_api_key: 実際のOpenAI APIキーに変更
   - paypal_client_id: 実際のPayPal Client IDに変更
   - paypal_secret: 実際のPayPal Secretに変更
   - preview_to: あなたのテスト用メールアドレスに変更
   - sender_email: 占い師用Gmailアドレスに変更
   - ops_email: 運営用メールアドレスに変更

2. links シート：
   - L9000: GASデプロイ後のURLに更新
   - L9001: 無料相談フォーム作成後のURLに更新
   - L9002: 無料鑑定フォーム作成後のURLに更新
   - L9003: 有料鑑定フォーム作成後のURLに更新

3. ai_prompts シート：
   - 各商品のプロンプトを実際の鑑定内容に合わせて調整

===============================================================================
本番運用時の設定変更
===============================================================================

テストが完了したら、config シートで以下を変更：

preview_mode: TRUE → FALSE
freeze_sending: FALSE（そのまま）
freeze_all: FALSE（そのまま）

これで本番運用が開始されます！

データ投入の順序（推奨）
===============================================================================

1. config シート（最優先）
2. ai_settings シート
3. ai_prompts シート
4. products シート
