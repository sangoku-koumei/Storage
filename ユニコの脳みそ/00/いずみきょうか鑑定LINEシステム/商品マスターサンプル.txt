■ 運用上の注意点
------------------------------------

【恋愛特化・溺愛型商品の強み】
✓ ターゲットが明確：「溺愛されたい女性」
✓ プロフィールとの一貫性：「我慢してきた恋→溺愛される恋へ」
✓ 投稿テーマとの一致：タロット、オラクル、深層心理学
✓ 価格で占術の量が増える分かりやすさ
✓ 「愛されている実感が湧く」という感情的ベネフィット
✓ 30,000円のみ恋愛以外も視る特別感

【占術の使い方】
✓ 西洋占星術：恋愛における性質を読み解く。価格で見る天体数が増える
✓ タロット/オラクル：彼の本音、未来、メッセージを引き出す
✓ 恋愛心理学：彼の行動パターンを科学的に分析
✓ 深層心理学：潜在意識のブロックを解除
✓ 溺愛スイッチ：いずみきょうか独自のメソッド
✓ 深層リーディング：オリジナリティと温かみ

【販売促進のヒント】
・無料鑑定で「愛されている実感」を少し体験してもらう
・5,000円商品は「彼の本心が知りたい」という切実なニーズに訴求
・10,000円商品は「溺愛されたい」という願望に直球アプローチ
・30,000円商品は「本気で変わりたい」覚悟を持った人へ
・サブスクは「一人で悩まなくていい」継続サポートの安心感
・30,000円商品は今後さらなる高額鑑定を作る可能性があるため「天体10個」と表記

【基本は恋愛特化、柔軟に対応】
✓ 依頼内容が恋愛以外（仕事、人間関係など）の場合も対応可能
✓ ただし30,000円以外は基本恋愛メイン
✓ 30,000円のみ恋愛以外の運勢も積極的に表記
✓ プロフィールは恋愛特化なので、集客も恋愛メインで
