## 🎯 クロージング（成約）テクニック

### テクニック1: 仮定法（If-Then）

```
「もし、このシステムを導入したら、
 3ヶ月後、あなたの月商は¥100万円超になります。
 
 もし、導入しなかったら、
 3年後も、今と同じまま。
 
 どちらを選びますか？」

→ 未来を2つ見せて、選ばせる
```

### テクニック2: 段階的合意（Yes Set）

```
「自由な時間が欲しいですよね？」
→ YES

「大きな収入も欲しいですよね？」
→ YES

「無料鑑定の自動化、実現したいですよね？」
→ YES

「では、このシステムが必要ですよね？」
→ YES（言わざるを得ない）
```

### テクニック3: 二者択一（Alternative Close）

```
「フル代行プランと、セルフ設定プラン、
 どちらがお好みですか？」

→ 「買うか買わないか」ではなく、「どっちを買うか」
```

### テクニック4: 今すぐ（Immediate Action）

```
「今日、お申込みいただければ、
 来週には納品できます。
 
 来週から、新しい人生が始まります。
 
 今日、決断してください。」

→ 今すぐ行動する理由を作る
```

### テクニック5: 最後の一押し（Final Push）

```
「最後に、お伝えします。
 
 私は、あなたに成功してほしい。
 
 本気で、そう思っています。
 
 だから、7日間の返金保証をつけました。
 だから、無制限のサポートをつけました。
 だから、月次レポートも提供します。
 
 あなたの成功に、本気でコミットします。
 
 （間・3秒）
 
 あとは、あなたが決めるだけです。
 
 今日、新しい人生を始めますか？
 
 それとも、明日も、無料鑑定に追われますか？
 
 （間・5秒）
 
 選択は、あなたのものです。」

【ポイント】
- 本気度を伝える
- 長めの間を取る
- 目を見る
- 最後は静かに
```

## 📊 営業的思考（セールスマインド）

### 営業の本質

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
営業とは、「売る」ことではない
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

営業とは、

「相手の問題を解決すること」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

だから、

1. 相手の問題を深く理解する
2. その問題を解決できる商品を提示する
3. 相手が成功するまでサポートする

これが、営業です。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### トップセールスの思考法

#### 思考1: Give First（先に与える）

```
「まず、価値を提供する」

- 無料資料
- 無料相談
- 無料ウェビナー

→ 先に与えると、返したくなる（返報性の原理）
```

#### 思考2: Win-Win（共に勝つ）

```
「相手が成功しないと、自分も成功しない」

→ だから、本気でサポートする
→ だから、返金保証をつける
→ だから、無制限のサポート
```

#### 思考3: Long Term（長期視点）

```
「1回売って終わりではない」

月額サポート = 長期的な関係

→ 顧客が成功すれば、継続してくれる
→ 紹介もしてくれる
→ LTV（顧客生涯価値）を最大化
```
