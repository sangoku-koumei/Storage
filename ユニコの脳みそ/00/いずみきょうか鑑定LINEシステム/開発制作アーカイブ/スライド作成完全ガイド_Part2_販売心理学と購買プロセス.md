# スライド作成完全ガイド Part2: 販売心理学と購買プロセス

**目的**: 占い師に「欲しい」と思わせ、行動させるための心理学的アプローチ

---

## 🧠 購買心理学の基礎

### 人が商品を買う理由（2つしかない）

```
1. 痛みから逃れたい（Pain）
2. 快楽を得たい（Pleasure）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

占い師の場合:

【Pain（痛み）】
❌ 無料鑑定に追われる苦しみ
❌ 時間がない不安
❌ 収入が増えない焦り
❌ 疲弊している現状
❌ このまま続けられない恐怖

【Pleasure（快楽）】
✅ 自由な時間
✅ 大きな収入
✅ 心の余裕
✅ 家族との時間
✅ 尊敬される存在

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

重要:
人は「Pleasure」より「Pain」から逃れたくて行動する

だから、まず「Pain」を強調する
```

## 🎯 AIDMAモデル（購買プロセス）

### A: Attention（注意）- 振り向かせる

**目的**: スクロールを止めさせる

```
【悪い例】
「占い師向けシステム販売中」

【良い例】
「SNS占い師のあなたへ、吉報です。
 無料鑑定に追われる日々、今日で終わりにしませんか？」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

テクニック:
✅ ターゲットを明確に（SNS占い師）
✅ 問いかけ形式
✅ 強烈なベネフィット（終わりにできる）
✅ 感情を刺激する言葉（吉報、追われる）
```

### I: Interest（興味）- 引き込む

**目的**: 「自分のことだ」と思わせる

```
【テクニック: 共感フレーズ】

「こんな毎日を送っていませんか？

 毎日、無料鑑定が山のように届く。
 1件1件、心を込めて鑑定する。
 でも、収入はゼロ。
 
 気づけば1日中、鑑定に追われている。」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

読んだ人の反応:
「そう！まさにこれ！」
「私のことを言っている…」

→ 続きを読みたくなる
```

### D: Desire（欲求）- 欲しくさせる

**目的**: 「これがあれば人生が変わる」と思わせる

```
【テクニック: 理想の未来を見せる】

「3ヶ月後のあなたを、想像してください。

 朝、ゆっくり起きる。
 コーヒーを飲みながら、スマホを見る。
 
 『昨夜、鑑定10件完了』
 『決済¥30,000完了』
 
 あなたは何もしていない。
 でも、収益は発生している。
 
 今日は、何をしようか。
 友達とランチ？
 カフェで読書？
 
 すべて、あなたの自由。
 
 この未来、手に入れたくありませんか？」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

脳科学:
人は「未来の自分」をリアルに想像すると、
それを実現したくなる

→ 具体的なシーンを描写する
→ 五感に訴える（コーヒーの香り、スマホを見る感覚）
```

### M: Memory（記憶）- 覚えてもらう

**目的**: 印象に残す

```
【テクニック: 数字のインパクト】

「月々¥23,100で、月¥230万円超」

この数字、忘れられますか？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【テクニック: キャッチフレーズの繰り返し】

スライド全体で、3回以上繰り返す:

「完全放置で月¥230万円超」
「完全放置で月¥230万円超」
「完全放置で月¥230万円超」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
同じメッセージを3回聞くと、記憶に残る
```

### A: Action（行動）- 今すぐ行動させる

**目的**: 申し込ませる

```
【テクニック: 限定性と緊急性】

「先着10名様限定」
「残り3枠」
「本日中のお申込みで特典」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【テクニック: リスクリバーサル（返金保証）】

「7日間、動作しなければ全額返金」

→ 購入のリスクを下げる

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【テクニック: 簡単な第一歩】

「まずは、無料相談から」
「資料請求だけでもOK」

→ ハードルを下げる
```

## 💰 価格の見せ方（心理学）

### テクニック1: アンカリング効果

```
【悪い例】
「¥198,000です」

【良い例】
「月商が30万円から80万円になると、
 年間で¥600万円の増加。
 
 その投資額は、わずか¥198,000。
 
 1ヶ月で回収できます。」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
大きな数字（¥6,000,000）を先に見せると、
¥198,000が安く感じる
```

### テクニック2: 分割の魔力

```
【悪い例】
「¥198,000です」

【良い例】
「1日あたり、わずか¥770円。
 
 コーヒー2杯分の価格で、
 月¥230万円の収入が手に入ります。」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

計算:
¥23,100/月 ÷ 30日 = ¥770/日

心理学:
大きな金額を、日割り・時給換算すると、
安く感じる
```

### テクニック3: 投資としての提示

```
【悪い例】
「費用は¥198,000です」

【良い例】
「投資¥198,000で、
 年間¥700万円のリターン。
 
 投資対効果：35倍。
 
 これは、投資です。」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
「費用」ではなく「投資」と表現すると、
価値が高く感じられる
```

### テクニック4: 損失回避

```
【テクニック】

「このシステムを導入しないと、
 あなたは今後も、
 毎月¥200万円を失い続けます。
 
 （月¥230万円 - 現状¥30万円 = ¥200万円）
 
 1年で¥2,400万円の損失。
 
 ¥198,000の投資をしないことで、
 ¥2,400万円を失う。
 
 これ、本当に良い判断ですか？」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
人は「得をする」より「損を避ける」方が強く動く
```

## 💰 価格提示の完璧なシナリオ

### ステップ1: 価値の積み上げ（Value Stacking）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
このシステムの価値は？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

「さて、このシステム。
 いくらだと思いますか？」

（間・聴衆に考えさせる）

「まず、開発コストから考えましょう。

 3年の歳月。
 1,000万円以上の投資。
 
 占いで稼いだお金、すべて。
 専門家への相談費用、数百万円。
 テストと改善、何百回。
 
 開発コストだけで、¥1,000万円以上です。」

（間）

「通常、このレベルのシステム開発を
 外注すると、¥300万円はかかります。」

（間）

「では、このシステムで得られる価値は？

 月¥230万円の収入。
 年間¥2,500万円。
 
 3年で¥7,500万円。」

（間・5秒）

「つまり、このシステムの本当の価値は、
 ¥7,500万円です。」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### ステップ3: 価格の正当化と特別価格

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
だから、決めました
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重要】スライドには金額を表示しない
【重要】口頭でのみ伝える

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

「本来の価格は、¥100万円。

 でも、今日だけ、
 
 （スライドを見せずに、口頭で）
 
 ¥198,000です。」

（間・5秒、驚きの表情を確認）

【ポイント】
- スライドには「今日だけの特別価格」とだけ表示
- 具体的な金額は口頭でのみ伝える
- 「¥100万円くらい」と思わせたまま、¥198,000を告げる
- 驚きと緊急性を最大化

 お話しします。」

（間）

「この価格は、
 『実績を作るため』の特別価格です。
 
 私は、データが欲しいんです。
 
 （真剣に）
 
 実際にシステムを使った占い師達が、
 どれだけ稼げるようになるのか。
 
 そのデータを集めたい。
 
 だから、今だけ、この価格です。」

（間）

「データが集まれば、
 販売価格は上がります。
 
 ¥50万円、¥100万円と。
 
 でも、今日申し込んでくださる方は、
 ずっと¥198,000のままです。」

（間・3秒）

「今日だけ、めいっぱい安くします。
 
 （笑顔で、親しみを込めて）
 
 ですから、皆さんが稼げるようになったら、
 ツールの維持のために、
 寄付でもしてくださいね。」

（会場、笑い）

（真剣に戻る）

「冗談ではなく、本当にそう思っています。
 
 皆さんの成功が、私の喜びです。」

（間・3秒）

「今日が、最安値です。
 
 明日には、値上げするかもしれません。」

- 実際の具体的金額はまだ明かさない

### ステップ7: 最終クロージング（最後の最後で具体的金額を口頭で）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ここまで、月10万円と思わせたまま進める
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重要】スライドには一切金額を書かない
【重要】最後の最後まで、口頭でも具体的金額は言わない

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

「さらに、7日間の返金保証。
 
 システムが動作しなければ、
 全額返金します。」

（間・相手は「でも稼げる保証はないしな…断ろうかな」と思い始める）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ここで、最後の最後に、口頭でのみ具体的金額を明かす
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

「では、具体的にお伝えします。
 
 （口頭でのみ）
 
 初期費用を5年分割で、月々約¥3,000。
 
 プロラインのプロプラン、月々¥9,800。
 これは、プロラインの会社に直接お支払いください。
 
 そして、毎月の維持費として、¥10,000。
 
 これは、先ほどお話しした
 エンジニアの雇用、
 システムの更新、維持、改善に
 必要になります。」

（間・3秒）

「その代わり、
 ○○の変更なら月○回まで、
 △△の変更なら月△回まで、
 対応させていただきます。」

（間・5秒、相手に計算させる）

「合計で、月々これだけです。」

（口頭でのみ具体的な金額を伝える）

（間・相手は「え？10万円じゃなくてこんなに安い！？それなら！」と思う）

【営業テクニック】
- 最後の最後まで「月10万円」と思わせて悩ませる
- 「高いな…でも稼げるなら…でも保証ないし…」と葛藤させる
- 断ろうかなと思い始めたところで、具体的金額を口頭で明かす
- 「こんなに安い！」と安堵と即決を促す
- すべて口頭でのみ。スライドには一切書かない

## 🎭 購買欲求を高める9つのトリガー

### 1. 希少性（Scarcity）

```
「先着10名様のみ」
「残り3枠」
「今月中のお申込み限定」

→ 「早くしないと、なくなる」と思わせる
```

### 2. 緊急性（Urgency）

```
「本日23:59まで」
「あと3時間で締切」
「期間限定特典」

→ 「今すぐ決断しないと」と思わせる
```

### 3. 社会的証明（Social Proof）

```
「すでに50名の占い師が導入」
「満足度98%」
「月商が3倍になった方も」

→ 「みんなが使っているなら安心」と思わせる
```

### 4. 権威性（Authority）

```
「月商80万円の実績システム」
「3,520行の本格的なコード」
「一般社団法人認定講座」

→ 「専門家が作ったなら信頼できる」と思わせる
```

### 5. 一貫性（Consistency）

```
「あなたは、占い師として成功したいですよね？
 （YES）
 
 そのためには、自由な時間が必要ですよね？
 （YES）
 
 では、このシステムが必要ですよね？
 （YES）」

→ 小さなYESを積み重ねる
```

### 6. 好意（Liking）

```
開発者ストーリーを語る:

「私も、あなたと同じでした。
 毎日、無料鑑定に追われて…」

→ 共感・親近感を持たせる
```

### 7. 互恵性（Reciprocity）

```
「まずは、無料で資料をプレゼント」
「30分の無料相談を実施」
「プロンプト集50種類を無料提供」

→ 「もらったから、お返ししないと」と思わせる
```

### 8. コントラスト効果

```
【Before】
1日12時間労働、月商30万円

【After】
1日10分作業、月商230万円

→ 差が大きいほど、魅力的に見える
```

### 9. 損失回避

```
「導入しないと、年間¥2,400万円を失います」

→ 「損したくない」という感情を刺激
```

## 🎯 PASONAの法則（販売の型）

### P: Problem（問題提起）

```
「無料鑑定100件、売上¥0。
 
 これが、あなたの現実ではありませんか？」

→ 問題を明確に提示
```

### A: Affinity（親近感）

```
「私も、同じでした。
 
 毎日毎日、無料鑑定をこなしていました。
 収入にならない作業を、延々と…」

→ 「この人は私のことをわかっている」と思わせる
```

### SO: Solution（解決策）

```
「そこで、私は考えました。
 
 『無料鑑定を、自動化できないか？』
 
 試行錯誤の末、完成したのが、
 このシステムです。」

→ 解決策を提示
```

### N: Narrow down（絞り込み）

```
「このシステムは、
 すでにSNS占い師として活動している方限定です。
 
 先着10名様のみ。」

→ 「自分は対象だ」「早くしないと」と思わせる
```

### A: Action（行動喚起）

```
「今すぐお申し込みください。
 
 あなたの人生が、3ヶ月後に変わります。」

→ 行動を促す
```

## 🧪 行動経済学（ナッジ理論）

### デフォルト効果

```
【悪い例】
「プランA、B、Cがあります」

【良い例】
「おすすめは、フル代行プラン（⭐推奨）です」

→ デフォルト（推奨）を示すと、それを選ぶ確率が上がる
```

### アンカリング効果

```
【価格の提示順】

1. 「通常、システム開発には¥300万円かかります」
2. 「でも、このシステムは¥198,000」
3. 「さらに、60回払いなら月々¥3,300」

→ 高い金額を先に見せると、後の金額が安く感じる
```

### フレーミング効果

```
【ネガティブフレーム】
「導入しないと、年¥2,400万円を失います」

【ポジティブフレーム】
「導入すると、年¥2,400万円を得られます」

→ 両方使うと効果的
→ まずネガティブで危機感、次にポジティブで希望
```

### 社会的規範

```
「すでに50名の占い師が導入しています」

→ 「みんながやっている」と思うと、安心して行動できる
```

### 現在バイアス

```
「今すぐ始めれば、3ヶ月後には月商¥100万円超」

→ 将来の利益を、今の決断と結びつける
```

```

「いえ、適正価格です」

【良い回答】
「確かに、¥198,000は大きな金額ですよね。
 
 でも、考えてみてください。
 
 このシステムで、月¥180万円の増収。
 年間¥2,400万円。
 
 つまり、1ヶ月で回収できます。
 
 2ヶ月目からは、すべて利益。
 
 ¥198,000の投資で、
 ¥2,400万円のリターン。
 
 これは、『高い』でしょうか？
 それとも、『安すぎる』でしょうか？」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

テクニック:
- 共感から入る（「確かに」）

- 問いかける（自分で気づかせる）
```

```
【良い回答】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 💎 メンタリズム（人心掌握）

### テクニック1: ミラーリング（鏡映し）

```
【スライド内で】

相手の言葉を、そのまま使う:

「無料鑑定に追われる」
「時間がない」
「収入が増えない」
「疲弊している」

→ 相手が使う言葉を使うと、共感が生まれる
```

### テクニック2: ペーシング（同調）

```
【スライドの流れ】

1. 相手の現状を語る（同調）
2. 相手の感情を語る（同調）
3. 相手の望みを語る（同調）
4. 解決策を提示（リード）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

最初は「Yes, Yes, Yes」と頷かせる
→ 最後の提案も「Yes」と言いやすくなる
```

### テクニック3: ダブルバインド（二者択一）

```
【質問の仕方】

悪い質問:
「買いますか？」

良い質問:
「フル代行プランと、セルフ設定プラン、
 どちらがお好みですか？」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
「買うか買わないか」ではなく、
「どっちを買うか」を選ばせる
```

### テクニック4: プライミング（事前準備）

```
【スライドの前半で】

「月¥230万円」という数字を、
何度も見せる。

→ 後半で価格¥198,000を見せると、
  「安い」と感じる

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

脳科学:
先に大きな数字を見せると、
後の数字の印象が変わる
```

### テクニック5: コミットメントと一貫性

```
【スライド中で、小さな合意を取る】

「自由な時間が欲しいですよね？」
→ 心の中で「Yes」

「大きな収入が欲しいですよね？」
→ 心の中で「Yes」

「それなら、このシステムが必要ですよね？」
→ 「Yes」と言わざるを得ない

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

心理学:
小さなYesを積み重ねると、
大きなYesも言いやすくなる
```

## 📊 データの見せ方

### 数字を印象的にする5つのテクニック

#### 1. 丸め（Rounding）

```
【悪い例】
「¥2,300,000」

【良い例】
「約¥230万円」または「月¥230万円超」

→ 丸めた数字は、覚えやすい
```

#### 2. 比較（Comparison）

```
「時給¥833 → ¥230,000
 
 276倍です。」

→ 比較すると、インパクトが強い
```

#### 3. 視覚化（Visualization）

```
グラフで見せる:

月商の推移:
¥30万 → ¥50万 → ¥100万 → ¥200万

→ 視覚は、理解を早める
```

#### 4. 累積（Accumulation）

```
「月¥180万円の増収。
 
 1年で¥2,160万円。
 3年で¥6,480万円。
 
 あなたが失うのは、これだけの金額です。」

→ 累積額は、インパクトが大きい
```

#### 5. 日割り（Daily Breakdown）

```
「月々¥23,100は、1日¥770。
 
 コーヒー2杯分です。」

→ 日割りにすると、安く感じる
```
