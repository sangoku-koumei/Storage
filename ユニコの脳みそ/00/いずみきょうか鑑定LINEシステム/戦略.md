# いずみきょうかインスタグラム戦略：無料診断→恋愛偏差値講座体験講座誘導戦略

## 📋 戦略の全体像

### ゴール
- **入口**: 無料診断への登録・参加（メルアド取得）
- **ゴール**: 恋愛偏差値講座の体験講座（無料）への参加・申込み

### 導線設計の基本方針（占い師アカウント向け）

```
インスタグラム投稿（占い師としての恋愛教育・日常発信）
    ↓
痛み・共感の蓄積（フォロワーとの信頼構築）
    ↓
公式LINE登録（リスト取り）
    ↓
LINEで話を進める（占い師としての恋愛教育・価値提供）
    ↓
「無料で自分がわかる診断、やってみてください」で無料診断へ誘導
    ↓
無料診断（メルアド必須）
    ↓
メールで教育
    - 最初は占い師としての話から始める
    - 段々「数値で見れる恋愛偏差値で可能な未来」を伝える
    - 「50〜100万円かけてもこれだけ学ぶのは難しい、時間も何年もかかる」
    - 「恋愛偏差値を理解して恋愛偏差値を上げる為に考えられた講座がある（社団法人日本ブライダルソムリエ協会）」
    - 「今回特別に無料で講座が受けられる（体験講座）」
    - 「無理な販売とかはしないので気軽に受けてみて、興味ある人は詳しく教えてと送って」
    ↓
体験講座（無料）への誘導
（値段は言わない）
    ↓
体験講座参加
```

## 🎯 セクション1：インスタグラムでの投稿戦略

### 1-1. 投稿の目的と設計思想

**前提**: 
- いずみきょうかのアカウントは**恋愛向けの占い師アカウント**として既に存在している
- 占い師としての信頼とフォロワーを活かす

**目的**: 
- 占い師として、フォロワーの「恋愛の痛み」を占いの視点から語り、共感を生む
- 「この占い師さんは、私のことを分かってくれる」という信頼を構築
- 公式LINEへの登録を促す（リスト取りが最優先）
- LINEで価値提供を続けてから、無料診断へ自然に誘導

**参考**: ノンオーガニック戦略.mdの「7日間教育」を参考に、SNS投稿でも段階的に教育する

## 🎓 セクション2：公式LINEでの教育（占い師としての恋愛教育）

### 2-1. 教育の目的と流れ

**目的**:
- 占い師として、恋愛に関する価値ある情報を提供し続ける
- 信頼関係を構築する
- 各ステップで反応をもらうことで、本気度の高い人を抽出
- 話がある程度進んだら、自然に無料診断へ誘導

**流れ**:
1. 公式LINE登録後、自動返信で「占い師としての恋愛の見えない構造」について少し触れる
2. ステップ1〜7で占い師としての教育（各ステップで簡単な反応を求める）
3. ステップ8〜14で恋愛偏差値の教育（各ステップで反応を求める。反応がない人は次に進まない）
4. ステップ15で講座の話 + 無料で受けれる話
5. その後は、いつでも「詳しく教えて」や「申し込み」などのキーワードで対応

**重要**: 
- すべてのステップで簡単な反応を求める（「ある」「いいえ」「1」「2」など）
- 反応がない人は次に進まない
- メルアドの有無にかかわらず、LINE教育で完結する

## 📱 セクション3：LINEでの体験講座対応設計（ステップ15以降）

### 3-1. LINEでの対応の目的と流れ

**目的**:
- ステップ15で「興味を持った人」が「体験講座」「詳しく教えて」「申し込み」などと送ってくる

- 合う人には自然に体験講座の日程を提案

**流れ**:

2. 自動返信で簡単な質問（または個別対応）

### 3-3. どちらのパターンを選ぶか（戦略の選択）

**パターンA（自動返信）：敷居を低くする設計**
- **メリット**: 
  - たくさんの応募が期待できる
  - 24時間対応可能
  - 運営の負担が少ない
- **デメリット**: 
  - 個別対応がないため、本気度の高い人も低い人も同じ対応になる
  - 体験講座の参加率が低くなる可能性がある
- **適用シーン**: まずは多くの人に体験講座を知ってもらいたい場合

**パターンB（個別対応）：本気の人を狙う設計**
- **メリット**: 
  - 本気度の高い人を抽出できる
  - 個別に状況を把握できるため、体験講座の参加率が高くなる可能性がある
  - 体験講座後の成約率も高くなる可能性がある
- **デメリット**: 
  - 運営の負担が大きい
  - 応募数が少なくなる可能性がある
- **適用シーン**: 質の高い応募を狙いたい場合

**推奨**: 最初は**パターンA（自動返信）**で始め、応募数と参加率を確認しながら、必要に応じて**パターンB（個別対応）**に切り替える
