## 📌 重要なポイント

### テスト期間中の設定

```
config シート:
preview_mode = TRUE（必須）
preview_to = あなたのメール

→ 実際のユーザーには送信されない
→ すべてpreview_toに届く
```

### 本番運用開始時の設定

```
config シート:
preview_mode = FALSE に変更

→ これだけで本番開始！
```

### 緊急停止

```
config シート:
freeze_all = TRUE に変更

→ すべての自動処理が停止
```
