# 占い師用 完全自動化システム

**LINE × PayPal × OpenAI × Google フォーム**

すべてが統合された、完全自動化の占いビジネスシステムです。

---

## 🎯 このシステムでできること

```
公式LINE登録
  ↓
プロラインのリッチメニューから申込み
  ↓
プロライン内でPayPal決済（有料の場合）
  ↓
Webhook → GAS自動処理
  ↓
相談内容フォームをメール送信（1回限り）
  ↓
ユーザーがフォーム記入
  ↓
OpenAI APIで鑑定書を自動生成
  ↓
メール自動送信
  ↓
完全自動化！
```

## 💡 システムフロー詳細

### 無料鑑定の場合

```
1. LINEリッチメニュー「無料鑑定」タップ
2. Googleフォーム表示（氏名、メール、生年月日、相談内容）
3. 送信
4. 受付メール送信（自動）
5. 翌日正午：選定通知メール
6. 同日夜：AI鑑定書生成＆メール送信
7. 24時間限定で無料相談の案内
```

### 有料鑑定の場合

```
1. LINEリッチメニュー「有料鑑定」タップ
2. PayPal.Me決済ページへ遷移
3. PayPal決済完了（¥5,000 or ¥10,000）
4. 運営がPayPal管理画面で確認
5. manual_payments シートに1行追加
   ↓（1分以内に自動）
6. users/payments/applications に記録
7. フォームトークン生成
8. メールで「相談内容フォーム」のURL送信
9. ユーザーがフォーム記入（1回のみ）
   ↓（翌日、自動）
10. OpenAI APIで鑑定書生成（3,500文字）
11. Google Driveに保存
12. 鑑定結果メール送信
```

### 無料相談の場合

```
1. LINEリッチメニュー「無料相談」タップ
2. Googleフォーム表示（候補日時3つ選択）
3. 送信
4. 運営に通知メール
5. 運営がZoom設定・日時決定
6. スプシに記入
7. 自動で確定通知＆リマインダー設定
8. Zoom面談実施
```
