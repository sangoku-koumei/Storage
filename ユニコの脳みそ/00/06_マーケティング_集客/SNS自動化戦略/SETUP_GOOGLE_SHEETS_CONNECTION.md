# Google Sheets 連携セットアップガイド

GASツール（スプレッドシート）とPythonツール（自動投稿・分析）を直結させるために、**Google Cloudの設定（無料）**が必要です。
少し手順が多いですが、これを一度だけ行えば、あとは全自動になります。

## 手順 1: Google Cloud プロジェクトの作成

1.  [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログインします。
2.  画面左上のプロジェクト選択リストをクリックし、「**新しいプロジェクト**」作成します。
    *   プロジェクト名: `SNS Automation` （なんでもOK）
    *   作成ボタンをクリック。

## 手順 2: APIの有効化

1.  プロジェクトが選択されていることを確認し、左メニューから「**APIとサービス**」>「**ライブラリ**」を選択。
2.  検索バーに `Google Sheets API` と入力して検索し、クリックして「**有効にする**」を押します。
3.  同様に `Google Drive API` も検索して、「**有効にする**」を押します。

## 手順 3: サービスアカウントの作成

1.  左メニューの「**APIとサービス**」>「**認証情報**」を選択。
2.  上部の「**認証情報を作成**」>「**サービス アカウント**」を選択。
3.  名前に `sns-robot` と入力し、「作成して続行」。
4.  ロール（権限）は「**編集者** (Basic > Editor)」を選択し、「完了」。

## 手順 4: 鍵（JSONファイル）の取得

1.  作成されたサービスアカウント（`sns-robot@...`）をクリック。
2.  上部の「**キー**」タブを選択。
3.  「**鍵を追加**」>「**新しい鍵を作成**」を選択。
4.  「**JSON**」を選んで「作成」。
5.  **ファイルが自動的にダウンロードされます。**
    *   このファイルを `service_account.json` という名前に変更してください。
    *   このファイルを、Pythonツールのフォルダ（`インスタコメント返信投稿予約` および `インスタ分析`）の中にコピーしてください。

## 手順 5: スプレッドシートの共有

1.  GASで使っている「SNS自動化」のスプレッドシートを開きます。
2.  右上の「**共有**」ボタンをクリック。
3.  「ユーザーやグループを追加」の欄に、**手順3で作成したサービスアカウントのメールアドレス**（`sns-robot@PROJECT_ID.iam.gserviceaccount.com`）を入力します。
    *   ※メールアドレスはJSONファイルの中にも `client_email` として書かれています。
4.  権限を「**編集者**」にして「送信」。

---

## 完了！
これで、Pythonツールがこのスプレッドシートの中身を読み書きできるようになりました。
ツールの設定に進んでください。
