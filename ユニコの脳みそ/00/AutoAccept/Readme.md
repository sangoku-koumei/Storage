# Auto Accept & Git Save Tool (Pro)

[[00_知識マップ|⬅️ 知識マップへ戻る]]

本ツールは、画面上の「Accept（承認）」ボタンを自動で検知してクリックし、同時にGitへの自動バックアップを行うプロフェッショナル向け自動化エンジンです。

## 🌟 パワーアップした機能

1.  **AIテキスト認識（UI Automation）**: 
    - 従来の画像認識を廃止し、Windowsシステム内部のテキスト情報を直接読み取ります。
    - **「含む（部分一致）」判定**: `TARGET_KEYWORDS` に "Accept" と一つ書いておくだけで、"Accept All" や "Accept Change" など、**その文字が含まれるすべてのボタン**に自動反応します。
    - **メリット**: ボタンが他の窓に隠れていたり、テーマ（ダークモード等）が変わっても、正確にクリック可能です。
2.  **多彩なキーワード対応**: 
    - `Accept`, `Accept All`, `Accept Change`, `Accept Alt`, `OK`, `Yes`, `承認` のすべてのバリエーションに自動対応。
3.  **Git自動バックアップ**: 
    - 15分ごとに作業環境を自動コミット。不慮の事故からデータを守ります。

## 🚀 使い方

### 1. 初回準備
以下のコマンドをターミナルで実行して、必要なエンジンをインストールしてください。
```powershell
pip install uiautomation
```

### 2. ツールを起動
フォルダ内の **`自動承認ツール起動.bat`** をダブルクリックするか、以下のコマンドを実行します。
```powershell
python auto_accept.py
```

### 3. ボタンのカスタマイズ
`auto_accept.py` 内の `TARGET_KEYWORDS` リストを編集することで、クリックしたい文字を自由に追加できます。
