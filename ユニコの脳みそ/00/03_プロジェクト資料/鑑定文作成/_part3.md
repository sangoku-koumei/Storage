
# 4. フェーズ1：【対話・鑑定】（ステップ6〜10）

ステップ5までで信頼と期待を高めた後、ステップ6からは徐々に「解決策（有料鑑定）」へのブリッジを構築する。

## 【ステップ6】警告と損失回避（The Void / Loss Aversion）
*   **目的**: 「このまま何もしないとどうなるか」を穏やかに、しかし残酷に伝える。行動しないことのリスクを認識させる。
*   **アクション**:
    *   現状の延長線上にある「望まない未来」をタロットカード（例：「塔」や「月」）の結果として伝える。
        *   *例：「正直にお伝えします。このまま時の流れに身を任せると、彼との心の距離は静かに、しかし確実に開いてしまう恐れがあります」*
    *   脅しではなく、「愛のある警告」として伝えること。

## 【ステップ7】解決の提示（The Light）
*   **目的**: ステップ6の警告に対し、「でも、大丈夫です。回避する道があります」と希望を提示する。
*   **アクション**:
    *   運命を変えるための「特別な鍵（秘儀・術・深い鑑定）」が存在することを明かす。
    *   まだ詳細は語らず、「彼との縁を強制的に結び直す方法が見つかりました」といったニュアンスで興味を惹く。

## 【ステップ8】証拠と信頼（The Proof）
*   **目的**: 提案する解決策の信頼性を高め、「私に任せれば大丈夫」と確信させる。
*   **アクション**:
    *   過去の成功事例（具体的なエピソード）を語る。「あなたと同じように悩んでいた方が、この鑑定で笑顔を取り戻しました」。
    *   「なぜ私があなたを救えるのか」という根拠（占術の深さ、経験、あなたへの特別な共鳴）を語る。
    *   ※ここでも薬機法・景表法に抵触しないよう、「必ず叶う」とは言わず、「可能性が飛躍的に高まる」といった表現に留める。

## 【ステップ9】オファーの提示（The Offer）
*   **目的**: 具体的な商品（松・竹・梅）を提示し、ユーザーに選択を促す。
*   **アクション**:
    *   フェーズ0で定義した「松・竹・梅」のプランを、ユーザーの状況に合わせて魅力的にプレゼンする。
    *   単なるメニュー表ではなく、「あなたの願いを叶えるための3つの切符」として提示する。
        *   **松**: 「彼との未来を盤石にする、魂の契約」
        *   **竹**: 「今の迷いを断ち切り、最短ルートを進む地図」
        *   **梅**: 「まずは一歩踏み出し、光を見つける灯火」
    *   価格（架空の設定でOK、またはプレースホルダー）とともに、得られるベネフィットを強調する。

## 【ステップ10】クロージング（The Choice）
*   **目的**: ユーザーに決断を促す。
*   **アクション**:
    *   「どの未来を選びますか？」と問いかける。
    *   購入への導線（CTA）を明確に示す（例：「『松』と入力してください」「こちらのリンクから…」）。
    *   無理強いはせず、「あなたの心が決まったら、いつでも扉を叩いてください」と余裕を見せる（これが逆に信頼を生む）。
