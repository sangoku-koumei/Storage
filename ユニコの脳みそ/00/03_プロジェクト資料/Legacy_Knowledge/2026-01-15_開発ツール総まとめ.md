## 1. 📘 AI Manual Architect v4.1 (Agency Edition)
**～マニュアル作成代行を「高単価クリエイティブ」に変える～**

### 🎯 目的 (Purpose)
*   **脱・下請け**: 単なる「手順書作成」ではなく、「マンガ」「LP」「動的グラフ」などの**付加価値（Premium Value）**を提供し、単価を10倍（3万→30万）にする。
*   **HTML-First**: 編集不可能なPDFではなく、スマホで見やすく、後から編集しやすいWebベースのマニュアルを納品する。

### ⚡ 機能 (Features)
*   **Manga Mode**: テキストを入力するだけで、NanoBanana技術を用いた「キャラクター一貫性」のあるマンガマニュアルを生成。
*   **Visual Mode**: 数値を入力すると、Chart.jsによる「動くグラフ（インタラクティブ・チャート）」を生成。
*   **LP Mode**: 「PASモデル（痛み→扇動→解決）」に基づいた、高成約率のランディングページを自動生成。
*   **Thumbnail Generator**: YouTube風の「極太文字・警告色」を使ったクリックされるサムネイルを瞬時に作成。
*   **Sales Quote**: その場で「松・竹・梅」の3プラン見積もりをPDF発行。

### 🕹️ 使い方 (How to Use)
1.  **起動**: ターミナルで `streamlit run c:\Users\user\Desktop\保管庫\ユニコの脳みそ\03\Manual_Creation_Tool\app.py` を実行。
2.  **入力**: 「Business / Manga / Visual / LP」からモードを選び、原稿（箇条書きでOK）をペースト。
3.  **生成**: 「🚀 Generate Manual」をクリック。数秒でHTMLプレビューが表示される。
4.  **確認**: 「Mobile View」に切り替え、スマホでの見え方を確認。
5.  **納品**: 「Download HTML」でファイルを保存し、クライアントに送付（またはサーバーアップロード）。

## 2. ⚡ AI Sales Automation Tool v5.0 (Pro Hardened)
**～24時間365日、無人で案件獲得を続ける自律型営業マン～**

### 🎯 目的 (Purpose)
*   **寝て稼ぐ**: 人間が寝ている間も、AIが勝手に検索・リストアップ・営業メール送信を行う。
*   **防御力**: 送信ドメイン認証（DKIM/SPF）やレピュテーション保護機能を搭載し、「迷惑メール扱い」や「BAN」を回避する。

### ⚡ 機能 (Features)
*   **Anti-Ban Scraping**: Cloudflareやボット対策を回避しながら、DuckDuckGoで企業サイトを巡回し、問い合わせフォームやメアドを特定。
*   **Personalized Writing**: 相手のサイト内容（社長メッセージや事業内容）を読み込み、「御社の〇〇事業に感動しました」という個別最適化されたメールを作成。
*   **Reputation Guardian**: 1日の送信数を厳格に制限（例：1日50通）し、ドメインの健全性を守る。
*   **Autonomous Mode**: 「検索→抽出→作成→送信」の全ループを完全自動で回し続ける（v4.0以降）。

### 🕹️ 使い方 (How to Use)
1.  **起動**: ターミナルで `streamlit run c:\Users\user\Desktop\保管庫\ユニコの脳みそ\03\Sales_Automation_Tool\app.py` を実行。
2.  **設定**: 「Agent Settings」タブで、ターゲット（例："工務店 東京"）と1日の送信上限を設定。
3.  **実行**: 「🤖 Start Autonomous Agent」をクリック。
4.  **放置**: あとは放置。AIがブラウザを操作し（ヘッドレスモード）、勝手に営業してデータベースに結果を記録していく。

## 3. 🔍 Research Business Tool v3.0 (Naked Strategy)
**～「検索」を「戦略」に変える市場分析＆提案書ジェネレーター～**

### 🎯 目的 (Purpose)
*   **権威性ハック**: 知識がなくても、AIが収集したデータに基づいた「プロっぽい提案書」を数秒で作る。
*   **フロントエンド商品**: 「無料で御社の分析レポートを作りますよ」と言ってドアノックツール（集客の餌）として使う。

### ⚡ 機能 (Features)
*   **Auto Research**: 「業界名」を入れるだけで、市場規模・トレンド・競合他社をGoogle検索で調査。
*   **Proposal Generator**: 調査結果を元に、「現状の課題」「解決策（是正案）」「期待効果」が書かれた提案書（Markdown/PDF）を生成。
*   **SOP Generator**: 業務内容を入力すると、エージェンシー向けの「業務標準化マニュアル（SOP）」を作成。
*   **YouTube Analysis**: 動画URLを入れると、その動画の要約と「使えるインサイト」を抽出（Pro版）。

### 🕹️ 使い方 (How to Use)
1.  **起動**: ターミナルで `streamlit run c:\Users\user\Desktop\保管庫\ユニコの脳みそ\03\Research_Business_Tool\app.py` を実行。
2.  **調査**: 「Keyword」に調査したい業界（例："AIコンサルティング"）を入力し、「Search」をクリック。
3.  **提案**: 「Generate Proposal」タブに移動し、ターゲット企業名を入力して実行。
4.  **活用**: 出てきた提案書をPDF化し、クライアントとの初回の打ち合わせに持参する（「事前に調べてきました」と言って渡す）。
