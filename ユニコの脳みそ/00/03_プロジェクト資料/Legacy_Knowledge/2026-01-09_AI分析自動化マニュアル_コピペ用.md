ユーザー様のご質問：
> **「3C分析、ベンチマーク(TTP)、アルゴリズム理解、SWOT分析、差別化戦略はツールで自動でやれるの？」**

**結論：はい、できます。**
ただし、「ボタンを1つ押したら全て完了」というツールは高額（月数十万円〜）なエンタープライズ向けが多いため、**「無料〜安価なAI（ChatGPT/Perplexity）に指示を投げて自動生成させる」**方法が個人の運用代行では「スパルタモード」として最適解です。

以下に、それぞれの分析をAIにやらせるための「コピペ用プロンプト（指示書）」をまとめました。これを使えば、**思考時間ゼロ**でプロレベルの分析レポートが出力されます。

## 0. 競合リサーチ（発掘）の自動化

そもそも「誰を真似すればいいかわからない」という場合、AIに探させます。
※ 検索機能付きAI（**Perplexity** / **ChatGPT Search** / **Gemini**）必須。

### 📋 コピペ用プロンプト
```markdown
#指示
私のビジネスの「競合」となりうる、SNS運用代行アカウント（またはサービス）を5つリストアップしてください。
以下の条件で検索し、具体的な「アカウント名（またはサービス名）」と「参考になるURL」を表形式で提示してください。

#検索条件
1. **ジャンル**: 個人〜小規模チームで活動している「Instagram/Threads運用代行」
2. **プラットフォーム**: Note, ココナラ, X(Twitter), Instagram
3. **ターゲット**: 低価格帯（月5〜10万）から高単価（月30万〜）まで幅広く

#出力項目
*   名前/サービス名
*   URL
*   推定単価（もし分かれば）
*   特徴・強み（一言で）
```

## 1. 3C分析 & 市場調査の自動化

市場（Customer）と競合（Competitor）をAIに調べさせます。
※ 最新情報が必要なため、検索機能付きAI（**Perplexity** または **ChatGPT Search**）を使用してください。

(上記でリストアップした競合URLを元に分析させます)

### 📋 コピペ用プロンプト
```markdown
#指示
あなたはプロのマーケターです。以下の業界・ジャンルにおける「3C分析」を行い、表形式で出力してください。
特に「顧客の悩み」と「競合の弱点」を鋭く分析してください。

#対象ジャンル
[ここにジャンルを入力：例：SNS運用代行（個人・フリーランス）]

#競合リスト（URLまたはサービス名）
1. [競合AのURL/名前]
2. [競合BのURL/名前]
3. [競合CのURL/名前]

#出力項目
1. Customer（市場・顧客）：ターゲット層の属性、深い悩み、成約に至るトリガー
2. Competitor（競合）：指定した3社の提供価値と、彼らが満たせていない「穴（不満点）」
3. Company（自社）：後発の我々が取るべき「強み」の仮説（競合の穴を突く提案）
```

## 2. ベンチマーク・TTP（徹底的にパクる）の自動化

「伸びている投稿」の共通言語をAIに抽出させます。
※ 実際の「URL」や「投稿テキスト」をAIに読み込ませると精度が爆上がりします。

### 📋 コピペ用プロンプト
```markdown
#指示
あなたはSNSアルゴリズムの解析エンジニアです。
以下に、同ジャンルで「伸びている投稿」のテキスト（またはキャプション）を貼り付けます。
これらを分析し、**「なぜ伸びたのか（アルゴリズム的に評価された点）」**を共通要素として抽出してください。

#分析対象データ
（ここに競合の伸びている投稿の文章やタイトルを3〜5個コピペする）

#出力項目
1. **トリガーワード**: ユーザーの手を止めた具体的な単語は？
2. **構成の型**: 結論から書いているか？共感から入っているか？
3. **TTPアクション**: 私が明日から投稿を作る際、具体的にどこを真似すればよいか？3つ指示してください。
```

## 3. アルゴリズム理解の自動化

「今のSNSで何がウケるか」はブラックボックスですが、AIは「一般論の最適解」を網羅しています。

### 📋 コピペ用プロンプト
```markdown
#指示
現在（2025-2026年）、[Instagram / Threads / X] のアルゴリズムにおいて、
「発見タブ（おすすめ）」に載るために最も重要視されている指標は何ですか？
最新のトレンド踏まえて、優先順位が高い順に5つ挙げ、それぞれに対して投稿者がやるべき「具体的なアクション」を1行で答えてください。
```

## 4. SWOT分析 & 差別化戦略の自動化

ここまでの情報を元に、「勝てる戦略」をAIに立案させます。これは「たたき台」として最強です。

### 📋 コピペ用プロンプト
```markdown
#指示
あなたは戦略コンサルタントです。
これまでの分析結果（3C、競合TTP）を元に、私のSNS運用代行事業の「SWOT分析」と「クロスSWOT戦略」を作成してください。

#私の状況（Input）
*   ジャンル：[例：インスタ運用代行]
*   強み：[例：AIを使った大量生産、レスが早い、価格が安い]
*   弱み：[例：実績が少ない、デザインはテンプレート頼み]

#出力内容
1. **SWOT分析表**: 強み・弱み・機会・脅威
2. **クロスSWOT戦略（重要）**:
    *   **積極戦略（強み×機会）**: 今すぐやるべき攻めの一手
    *   **差別化戦略（強み×脅威）**: 競合に勝つための「一点突破」ポイント
    *   **改善戦略（弱み×機会）**: 弱みをどうカバーして波に乗るか
3. **キャッチコピー案**: 私のプロフィールに書くべき「差別化された肩書き」を3案
```

## 5. 自動化ツール・スタック構成

上記は「AIチャット」を使った半自動化ですが、さらに機械的に行うなら以下の構成推奨です。

| 分析フェーズ | 推奨ツール（低コスト） | 自動化レベル |
| :--- | :--- | :--- |
| **競合リサーチ** | **Notion Web Clipper** | **Lv.1** (手動収集→AI保存) |
| **投稿分析** | **ChatGPT (画像解析)** | **Lv.2** (スクショを貼って「改善点教えて」と聞くだけ) |
| **レポート化** | **Gamma / Claude** | **Lv.3** (テキストを渡して「パワポ資料にして」と頼むとスライドが完成) |

### 運用イメージ（スパルタモード）
1.  **朝**: スマホで競合の「良い投稿」を見つけたらスクショ or URLコピー。
2.  **昼**: GPTに「これ分析してTTPポイント教えて」と投げる。
3.  **夕**: 出てきた「勝ちパターン」に自分のネタを当てはめてCanvaで作る。

このサイクルなら、難しいフレームワークを勉強する必要はありません。**「AIに聞くクセ」をつけることこそが、最強の自動化です。**
