# n8n版 Threads自動化 構築プラン

[[00_知識マップ|⬅️ 00：知識マップへ戻る]]

## Keywords & Tags
#n8n #Threads #Automation #VPS #GAS #Legacy_Knowledge #00

ユーザー様のご指摘通り、Pythonツールは**「PCの電源が入っていないと動かない」**という弱点があります。
夜間や外出中も勝手に動いてほしい場合、**n8n（ノーコードツール）** で構築するのが正解です。

ご要望の「Threadsコメント自動返信」および「自動投稿」をn8nで再現する設計図を作成しました。

---

## ⚠️ 重要な前提（サーバーについて）

n8nで「PC電源OFFでも稼働」させるには、以下のいずれかが必要です。
1.  **n8n Cloud (有料)**: 月額約3,000円〜。一番楽。
2.  **VPS自前構築 (格安)**: 月額500円〜（Xserver VPSなど）。スパルタ推奨。
3.  **デスクトップ版**: **結局PCの電源を入れる必要があります**（Pythonと同じ）。

※もし「完全無料がいい」場合は、**GASのみで実装する**のが唯一の道ですが、今回はご指定の **「n8n」** で設計します。

## 🛠️ n8nワークフロー設計

### 1. Threads 自動返信フロー (Auto-Reply)

**Trigger: Schedule Trigger (例: 10分おき)**
※Threads APIはまだWebhook（リアルタイム通知）が不安定なため、定期巡回が確実です。

1.  **HTTP Request**: Threads API `GET /me/threads?fields=id,text,comments`
    *   自分の最新投稿と、それについたコメントを取得。
2.  **Code (Filter)**:
    *   「まだ返信していない」かつ「特定のキーワード（詳細など）を含む」コメントを抽出。
3.  **OpenAI (ChatGPT)**:
    *   コメント内容に対して「返信文」を生成させる（またはスプシの定型文を取得）。
4.  **HTTP Request**: Threads API `POST /media/threads`
    *   リプライを投稿。

### 2. インスタ・Threads 同時投稿フロー (Auto-Post)

**Trigger: Schedule Trigger (例: 毎日 19:00)**

1.  **Google Sheets**: `final_schedule.csv` (またはシート) を読み込み。
    *   「今日の日付」の行を検索。
2.  **Filter**: 「今日の投稿データ」があるか確認。
3.  **HTTP Request (Instagram)**:
    *   画像URLとキャプションを使ってフィード投稿。
4.  **HTTP Request (Threads)**:
    *   同じ内容をThreadsにも投稿。
5.  **Google Sheets**:
    *   ステータスを「Posted」に更新。

## 📝 導入ステップ

### Step 1: 環境選び
Python（PC常時起動）がNGであれば、**n8nをどこで動かすか**決める必要があります。
*   もし「月500円くらいならOK」であれば、**VPSでのn8n運用**をおすすめします。これが最もコスパ良く高機能です。

### Step 2: ワークフロー作成
上記の設計に基づき、n8nの画面でノードを繋いでいきます。
もしn8nの環境が既にお手元にある（あるいはご存知のツールがある）なら、**「JSONコード（ワークフローの設計図）」**をお渡しすれば、インポートするだけで即完成します。

### Step 3: API連携
ThreadsとInstagramの「アクセストークン」をn8nのCredentialsに登録します。
