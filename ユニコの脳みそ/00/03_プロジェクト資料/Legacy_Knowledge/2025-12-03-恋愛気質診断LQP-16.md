# 恋愛気質プロファイルLQP-16メモ

## Keywords & Tags

## 0. 診断のコンセプト全体
### 診断名
- 正式名称：恋愛気質プロファイル16  
- 略称：LQP-16（Love Quotient Profile 16）

### コンセプト
- MBTIの構造を参考にしつつ「恋愛に特化した性格診断」。  
- 心理学（愛着スタイル、ビッグファイブ）、行動学（依存・回避行動）、脳科学（報酬系・新奇性追求）がベース。  
- 恋愛体質を4つの軸でプロファイル化。

### 想定ユーザー
- 10代〜30代の恋愛・婚活層。  
- 恋愛傾向・相性・向いている恋愛スタイルを知りたい人。

### 出力内容
- 4文字コード（例：EDFA）  
- タイプ名（正式＋短縮）  
- キャッチコピー  
- タイプ説明（特徴／得意な恋愛／こじれポイント）  
- 相性の良いタイプ／注意タイプ

## 1. 恋愛気質を測る4つの軸
| 軸 | ラベル | 説明 |
| --- | --- | --- |
| ① 愛情表現スタイル | E / I | 表出型 vs 内省型 |
| ② 依存度・距離感 | D / S | 溺愛型 vs 自立型 |
| ③ 感情処理スタイル | F / R | 感情優位型 vs 理性優位型 |
| ④ 恋愛モチベーション | A / X | 安定志向 vs 刺激志向 |

### 軸詳細
1. **愛情表現スタイル：E / I**  
   - E（表出型）：感情・好意を外に出しやすい。社会的報酬への感度が高い。  
   - I（内省型）：内側で強く感じるが表現は控えめ。内的世界を重視。

2. **依存度・距離感：D / S**  
   - D（溺愛型）：相手中心。返信や態度に敏感。不安型愛着寄り。  
   - S（自立型）：自分の軸を保つ。恋愛と他領域を切り分けやすい（安定〜回避寄り）。

3. **感情処理スタイル：F / R**  
   - F（感情優位型）：感じたことを最優先。情動系が強い。  
   - R（理性優位型）：事実・ロジック重視。前頭前野で状況整理。

4. **恋愛モチベーション：A / X**  
   - A（安定志向）：安心感・継続性がモチベーション。  
   - X（刺激志向）：新鮮さ・変化がモチベーション。

## 2. 診断質問（サイト用テキスト）
### 回答ルール
- 各質問ごとに「A」「B」どちらが自分に近いかを選択。  
- 完全に判断できない場合は、より近い方を選ぶ。  
- 質問は全20問。  
- 4軸ごとにA/Bの数を集計して判定。

#### ① 愛情表現スタイル（E / I）
- Q1-1: A 好意をすぐ表す / B 確信まで出さない  
- Q1-2: A 人前スキンシップ平気 / B 二人きりで十分  
- Q1-3: A 連絡頻度は多い方 / B 用事があるときだけ  
- Q1-4: A 予定が増えるとテンションUP / B 増えすぎると疲れる  
- Q1-5: A 言葉で伝えるべき / B 行動や空気で伝わればOK

#### ② 依存度・距離感（D / S）
- Q2-1: A 返信が遅いと気になる / B 忙しいと流せる  
- Q2-2: A 恋人優先しがち / B 自分の予定も大事  
- Q2-3: A 相手の感情変化に敏感 / B 「今はそういう時期」と流す  
- Q2-4: A ハマると頭がいっぱい / B 切り替えやすい  
- Q2-5: A 相手に合わせて自分を変えがち / B できるだけ軸を守りたい

#### ③ 感情処理スタイル（F / R）
- Q3-1: A 感情から想像 / B 事実ベースで推理  
- Q3-2: A ケンカは感情を分かってほしい / B 何が問題か整理したい  
- Q3-3: A 一言で引きずることがある / B 理由が分かれば切り替え  
- Q3-4: A ロマンチックに弱い / B 具体行動や計画が響く  
- Q3-5: A 感情が追いつかないこと多い / B 現実的にどうするか考える

#### ④ 恋愛モチベーション（A / X）
- Q4-1: A 安心できる関係が良い / B ときめきや新鮮さが大事  
- Q4-2: A 一人とじっくり / B いろいろな人と出会いたい  
- Q4-3: A 穏やかな恋が好き / B 波乱も醍醐味  
- Q4-4: A 予定が読めると安心 / B 急な誘いがワクワク  
- Q4-5: A 将来設計が気になる / B 今どれだけ楽しいかが重要

---

## 3. 判定ロジック（実装）
1. 全20問に回答。  
2. 各軸でA/Bの数をカウント。  
   - 軸①（Q1-1〜1-5）：A多→E、B多→I  
   - 軸②（Q2-1〜2-5）：A多→D、B多→S  
   - 軸③（Q3-1〜3-5）：A多→F、B多→R  
   - 軸④（Q4-1〜4-5）：A多→A、B多→X  
3. 4つの結果を連結してタイプコード作成（例：ESFX）。  
4. タイプコードをキーに、正式名称・短縮名・キャッチコピー・説明文・相性データを引き当てる。

### 同数時の仕様（案）
- デフォルトを決める（例：E/S/R/A寄せ）  
- または「迷う場合はよりなりたい方を選ぶ」と明記して同数発生を減らす。

## 5. 相性表の使い方
- 各タイプごとに「相性◎」「相性○」「注意△」を設定。  
- 結果ページには少なくとも相性◎（3タイプ前後）と注意△（2〜3タイプ）を表示。  
- ロジック概要：  
  - 依存度が高いDタイプは安定・堅実（A／R寄り）と◎。  
  - 刺激・自由系（X／ESRX／EDRX／ISRXなど）は同系統とは◎、安定依存系とは△。  
  - 似た価値観は○〜◎、正反対は△。
