## 🛠️ 1. Automation_Tools（SNS運用・総合ハブ）

`streamlit run app.py` で起動するダッシュボードから、以下の全ツールにアクセス可能です。

### 📊 【リサーチ・戦略】
- **01 全コンテンツ競合リサーチ (Naked Strategy)**
  - **機能**: YouTube、SNS、ココナラの競合URLを入力するだけで、彼らの「強み・弱み・稼ぎの源泉」をAIが分析。
  - **使い方**: 競合情報とAPIキーを入力し「リサーチ実行」。結果は自動保存され、次の投稿作成の材料になります。
- **07 自動鑑定書・商品作成 (Fortune Product Forge)**
  - **機能**: 占いジャンルの「売れる商品」を自動生成。マルチエージェント会議を行い、鑑定書の型まで作成します。

### ✍️ 【コンテンツ制作】
- **02 投稿作成ツール**
  - **機能**: 01のリサーチ結果に基づき、読者の心を動かすステップ投稿やバズるツイート、Instagramキャプションを生成。
- **05 画像自動生成 (DALL-E 3)**
  - **機能**: 抽象デザインから実写風まで、高クオリティなサムネイルをワンクリックで量産。
- **06 画像自動リネーム**
  - **機能**: フォルダ内の乱雑な画像ファイルを、SEOに適した分かりやすい名前に一括処理。

### 🤖 【運用・自動化】
- **03 Insta Ghost (Instagram Stealth Post)**
  - **機能**: 公式APIによる「Auto Mode」と、安全な手動投稿をQRコードで支援する「Safe Mode」の切り替えが可能。
- **04 自動コメント返信**
  - **機能**: 投稿へのコメントに対し、ファンの熱量を最大化する「神対応レスポンス」を自動生成。
- **Special_Appraisal_Master**
  - **機能**: 占いの個別鑑定に特化。心理学とスピリチュアルを融合した、リピートを生む鑑定文を生成。
