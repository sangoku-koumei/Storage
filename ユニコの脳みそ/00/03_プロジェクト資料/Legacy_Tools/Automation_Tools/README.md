## 🚀 起動方法

1. **PowerShell (ターミナル)** を開きます。
2. 以下のコマンドを入力して実行します。
   ```powershell
   cd "c:\Users\user\Desktop\保管庫\ユニコの脳みそ\Automation_Tools"
   streamlit run app.py
   ```
3. もし `Email:` と聞かれたら、なにも入力せずに **Enterキー** を押してください。
4. ブラウザが自動的に開き、ツールの画面が表示されます。

## 🔑 準備（APIキー）

このツールは高度なAI分析を行うため、以下のAPIキーが必要です。これらはツールのサイドバー（画面左側）に入力欄があります。

- **OpenAI API Key**: ChatGPT (GPT-4) や画像生成 (DALL-E 3) に使用します。
- **YouTube Data API Key**: YouTubeの動画やコメントを検索・取得するために使用します。

---

## 🛠 各ツールの使い方

画面左側のサイドバーメニュー（隠れている場合は `>` ボタンをクリック）からツールを選んでください。

### 01 全コンテンツ競合リサーチ (Naked Strategy)
競合の「勝ち筋」を丸裸にするツールです。
1. **API Key設定**: サイドバーにYouTube API Keyを入力。
2. **キーワード入力**: 「溺愛」「復縁」など、リサーチしたい単語を入力。
3. **リサーチ開始**: ボタンを押すと、YouTube上の人気動画と、そのコメント欄（見込み客の悩み）を取得して表示します。

### 07 自動鑑定書・商品作成 (Fortune Product Forge)
売れる占い商品を全自動で企画・作成します。
1. **API Key設定**: サイドバーにOpenAI API Keyを入力。
2. **テーマ入力**: 作りたい商品のテーマ（例：「不倫」「金運」）を入力。
3. **会議開始**: ボタンを押すと、4人のAI専門家が議論を開始し、「タイトル」「商品文」「サムネイル画像」まで全自動で生成します。

### 05 画像自動生成
SNS投稿用の画像を生成します。
1. **API Key設定**: OpenAI API Keyを入力。
2. **プロンプト**: 「どんな画像を作りたいか」を日本語で入力。
3. **生成**: 数秒で高画質な画像が生成されます。右クリックで保存して使ってください。

### 06 画像自動リネーム
溜まってしまった画像ファイルを整理します。
1. **対象フォルダ**: リネームしたい画像が入っているフォルダのパスを入力。
2. **プレフィックス**: ファイル名の頭につけたい文字（日付など）を入力。
3. **実行**: フォルダ内の全画像が一括で連番付きの名前に変わります。

## 🌪️ 深層対話：自動化スイートが描く「孤独な富豪」の設計図

**テーマ**: 各種ツールの断片を統合し、自律的な収益サイクルを構築する

**参加者**:
*   **Strategist**: Naked Strategy（競合分析）を武器に、市場の富を刈り取る軍師。
*   **Implementer**: 各種自動化スクリプトの安定稼働と、確実なアウトプットを管理する。
*   **AI Architect**: Fortune Product Forge（商品生成）のプロンプトエンジニアリング担当。
*   **Unico (PM)**: ツールたちが「一つの生命体（Agent）」として機能するよう統括する。

### 第1章：なぜ、バラバラのツールを一箇所にまとめたのか

**Strategist**: 
かつて、分析は分析、画像生成は画像生成と、ツールが散乱していました。しかし、それでは「リサーチから商品化まで」の速度が出ません。

**Implementer**: 
`🚀 起動方法` にあるように、Streamlit 1つ立ち上げるだけで、左側のサイドバーですべてを切り替えられる。この「司令塔」としての設計こそが、この README の核です。

**AI Architect**: 
「01 全コンテンツ競合リサーチ」で YouTube から見込み客の「悲鳴（悩み）」を拾い、その直後に「07 自動鑑定書作成」でその悩みを解決する。この流れるような連携が、我々のいう「スイート」の正体です。

**Unico**: 
**【提言1】 ツールを「点」で使うな。「線（フロー）」で繋げ。**
分析・企画・生成・リネーム。この一連の流れを一つの README に収めることで、作業の断絶（コンテキストスイッチ）を最小化しています。

### 第2章：APIキーという「血液」の循環

**Implementer**: 
`準備（APIキー）` セクション。ここが動脈ですね。OpenAI と YouTube API。これらが枯渇すると、私たちの「脳」は動かなくなります。

**AI Architect**: 
特に OpenAI キーは DALL-E 3（画像生成）も兼ねています。`05 画像自動生成` で生成した「売れる商品画像」を、そのまま `06 画像自動リネーム` で整理する。ここでも連携が重要です。

**Strategist**: 
投資すべきは「自分の労働」ではなく、これら API という「他者の労働力」です。

### 第3章：「Naked Strategy」：暴かれるのは競合か、自分か

**Strategist**: 
`01 Naked Strategy` 機能。競合を丸裸にすればするほど、実は自分のインプットの浅さに気づくはずです。

**AI Architect**: 
YouTube のコメント欄から拾った「生きた言葉」を、そのまま Fortune Product Forge のプロンプトに流し込む。AIに嘘をつかせないための、最良のエビデンスがここにあります。

**Unico**: 
それが `2026-01-13_ツール開発知見バイブル` での発見でした。「AIに事実を食わせる」こと。
