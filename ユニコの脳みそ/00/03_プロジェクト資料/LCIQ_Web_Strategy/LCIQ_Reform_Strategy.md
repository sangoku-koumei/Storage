# LCIQ Web戦略・改善提案書

## 1. 現状分析
### 良い点（資産）
*   **強いオファー**: 取扱店登録料 22,000円、年会費無料、セミナー無料など「始めやすさ」が明確。
*   **実績の証明**: 累計受講者数・活動エリア・行政イベント事例・受講者の声など信頼の証拠が揃っている。
*   **導線**: 無料セミナーや問い合わせへの受け皿が用意されている。

### 改善余地（課題）
*   **ターゲットの混在**: 「診断を受けたい一般ユーザー」と「ビジネスで扱いたい取扱店希望者」が混ざっており、初見で迷いやすい。
*   **信頼性の阻害要因**: 「成婚率90%」「楽々集客」などの強い数字が、根拠・注釈なしで並ぶと逆に怪しく見える。
*   **フォームのハードル**: 住所必須など入力項目が多く、離脱要因になっている。
*   **FAQ不足**: 反論処理（不安の解消）が足りない。

## 2. 全体戦略：2つの入口（迷子ゼロ化）
TOPページでターゲットを明確に分岐させる。

*   **A：自分の恋愛コミュ力を知りたい（一般ユーザー）** → 恋愛力アップLPへ
*   **B：ビジネスで活用したい（講師・取扱店）** → 講師募集LPへ

## 3. 改善版ターゲット設定
### ① 本人の恋愛力アップ（メイン）
*   **ターゲット**: 恋活・婚活中、またはパートナーとの関係に悩む人。
*   **ゴール**: 無料診断 → 無料セミナー/詳細診断 → 有料プログラム等
*   **訴求**: 「気合い」ではなく「自分のクセ（偏差値）」を知ることで、無駄な努力を減らす。

### ② 講師を目指す（ビジネス本命）
*   **ターゲット**: 恋愛相談に乗ることが多い人、教える仕事がしたい人。
*   **ゴール**: 無料診断/セミナー → 講師コース（短期集中・試験あり）
*   **訴求**: 「感覚」ではなく「体系化された理論」で教える武器を手に入れる。

### ③ 取扱店（サブ／取りこぼし救済）
*   **ターゲット**: LCIQ診断を集客ツールとして使いたいが、講師までは目指さない人。
*   **位置づけ**: LPの最後に控えめに案内。「ツール利用権」として訴求。

## 4. 信頼設計（怪しさの払拭）
*   **数字の根拠**: 実績数字には必ず「※（注釈）」を入れる（期間、対象、個人差など）。
*   **当日限定特典**: 値引きなどの安易な煽りではなく、「学習計画サポート」「導線設計アドバイス」など誠実な特典で背中を押す。
*   **2択CTA**: 「診断」or「セミナー」など、ユーザーがその場で選びやすい選択肢を提示し、「追いメールなし」でもCVさせる。

## 5. 推奨サイトマップ
1.  **TOP**: 「あなたはどちらですか？」で分岐
2.  **LP A (一般向け)**: 恋愛偏差値診断 → 無料セミナー/詳細診断予約
3.  **LP B (ビジネス向け)**: 講師募集（＆取扱店案内） → 無料セミナー/説明会予約
4.  **共通**: 実績・事例（注釈付き）、FAQ、企業情報（特商法）
