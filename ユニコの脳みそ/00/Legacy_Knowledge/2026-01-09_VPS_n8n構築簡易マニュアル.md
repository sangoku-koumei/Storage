# VPS × n8n 構築簡易マニュアル

Backlink: [[2026-01-09_作業メモ]] [[00_Deep_Knowledge_Index]]

## Keywords & Tags
#VPS #n8n #Infrastructure #Manual #Legacy_Knowledge

月額500円〜のVPSサーバーに、最強の自動化ツール「n8n」をインストールする手順です。
（※ Xserver VPS / Ubuntu 22.04 を想定していますが、他社VPSでもほぼ同じです）

---

## 🏗️ Step 1: VPSの契約 & SSH接続

1.  **契約**: Xserver VPS (2GBプラン以上が推奨ですが、512MB/1GBでも動きます) を契約。
    *   OSイメージ: **Ubuntu 22.04** を選択。
2.  **接続**: ターミナル（TeraTermやPowerShell）からSSH接続します。
    ```bash
    ssh root@<VPSのIPアドレス>
    # パスワードを入力してログイン
    ```

---

## 🏗️ Step 2: Dockerのインストール

n8nを動かすための「コンテナ基盤」を入れます。以下のコマンドを1行ずつコピペしてください。

```bash
# パッケージ更新
apt update && apt upgrade -y

# Dockerインストール
apt install docker.io -y

# Docker Composeインストール
apt install docker-compose -y

# 動作確認
docker --version
# → Docker version 20.x.x... と出ればOK
```

---

## ⚡ Step 3: n8nの起動

魔法の呪文（Dockerコマンド）を唱えて、n8nを立ち上げます。

```bash
# n8nを起動 (ポート5678で待受)
docker run -d \
  --name n8n \
  -p 5678:5678 \
  -e N8N_BASIC_AUTH_ACTIVE=true \
  -e N8N_BASIC_AUTH_USER=admin \
  -e N8N_BASIC_AUTH_PASSWORD=password \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```

*   `admin`: ログインID（自由に変更可）
*   `password`: ログインパスワード（**必ず変更してください！**）

---

## 🌐 Step 4: ブラウザからアクセス

1.  ブラウザを開き、`http://<VPSのIPアドレス>:5678` にアクセス。
2.  ID/Passを聞かれるので、先ほど設定したものを入力。
3.  **n8nの編集画面が表示されれば成功です！** 🎉

---

## 📝 補足: 独自ドメイン & SSL化 (上級編)

そのままだと `http://` (鍵マークなし) のため、スマホアプリ等からのWebhookが失敗することがあります。
本格運用するなら、**Cloudflare Tunnel** や **Nginx + Let's Encrypt** でHTTPS化するのが推奨されます。

まずは **「httpで動くこと」** を確認してから、必要に応じて設定しましょう。
（※最初はhttpでも、Instagram APIへのポーリング等は問題なく動きます）
