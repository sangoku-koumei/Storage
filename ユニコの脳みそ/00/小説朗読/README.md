# 小説朗読：アーカイブ (Archive)

[[../00_知識マップ|⬅️ 知識マップへ戻る]]

## 概要
「小説家になろう」などのWeb小説を音声化するツールと、生成された音声ファイルのアーカイブです。

## 📂 主なファイル
- **[[2026-01-05_小説朗読]]**: 開発ログとツール仕様。
- **[[novel_audio_maker.py]]**: 音声化スクリプト本体。
- `Master_Library/`: 全オーディオブックの保存先。
- `For_iPhone/`: 転送用フォルダ。

※ ファイル数が膨大（4000以上）なため、個別のリンクは作成していません。
