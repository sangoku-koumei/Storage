## 📱 ツール概要
「なぜ伸びないのか？」をデータで可視化するための分析ダッシュボード。Streamlitを用いたGUIで、自分と競合の違いを徹底的に比較する。

### 主な機能
1.  **自社データ収集 (Graph API)**
    *   いいね、コメント、保存、リーチなどのインサイトデータを自動収集。
    *   正確で安全な公式API利用。
2.  **競合分析 (Instaloader)**
    *   競合アカウントの投稿データを収集。
    *   「バズった投稿」と「普通の投稿」の違いを可視化。
3.  **比較・可視化 (Streamlit)**
    *   箱ひげ図などで「いいね数」「文字数」「ハッシュタグ数」の相関をグラフ化。
4.  **AI分析プロンプト生成**
    *   CSVデータをChatGPTに分析させるための「プロンプト」を自動発行。

## 🛠 技術スタック
*   **言語**: Python
*   **GUI**: Streamlit
*   **収集**: Instagram Graph API, Instaloader, Selenium (動画再生数取得用)
*   **特徴**: AI非搭載（ChatGPT連携前提）、CSV/Excel出力

## 🔄 GASツールとの連携（Feedback Loop）
本ツールは、運用結果を分析し、GASツールの精度を高める役割を持つ。

1.  **GAS**: コンテンツ生成・投稿。
2.  **運用**: 1ヶ月継続。
3.  **本ツール**: 「どの投稿が伸びたか？」「競合のトレンドは？」を分析。
4.  **改善**: 分析結果に基づき、GASの `PERSONA_SETTING`（世界観）や `systemPrompt` を修正する（例：「もっと辛口の方が保存数が多い」など）。
