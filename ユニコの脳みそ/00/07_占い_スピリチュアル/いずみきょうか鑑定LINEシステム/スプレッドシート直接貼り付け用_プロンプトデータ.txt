====================================
スプレッドシート直接貼り付け用データ
====================================

【超簡単セットアップ手順】

1. Googleスプレッドシートで新しいシート「ai_prompts」を作成
2. A1セルを選択
3. 以下のヘッダー行をコピーして貼り付け
4. 各プロンプトを1行ずつコピーして貼り付け

====================================
■ ヘッダー行（A1セルに貼り付け）
====================================

prompt_id	prompt_type	product_id	title	content	active	sort_order	notes

====================================
■ プロンプトデータ（A2セルから順に貼り付け）
====================================

【重要】
各プロンプトは非常に長いため、Eカラム（content）に直接入力する方法を説明します。

====================================
手順：
====================================

1. まず基本情報だけ貼り付け（A列〜D列、F列〜H列）
2. 後でE列（content）に各プロンプトファイルの内容を貼り付け

====================================
■ 基本情報（A2〜H2に貼り付け）
====================================

prompt_free_001	love_reading	FREE_001	溺愛され体験無料鑑定プロンプト	（後で入力）	TRUE	1	無料鑑定・溺愛体験（500文字）

====================================
■ 基本情報（A3〜H3に貼り付け）
====================================

prompt_paid_5000	love_reading	PAID_5000	彼の本心リーディング鑑定プロンプト	（後で入力）	TRUE	2	有料鑑定5000円・彼の本心（2000-3000文字）

====================================
■ 基本情報（A4〜H4に貼り付け）
====================================

prompt_paid_10000	love_reading	PAID_10000	溺愛スイッチ発動鑑定プロンプト	（後で入力）	TRUE	3	有料鑑定10000円・溺愛スイッチ（5000-8000文字）

====================================
■ 基本情報（A5〜H5に貼り付け）
====================================

prompt_paid_30000	love_reading	PAID_30000	溺愛体質完全変換・運命転換鑑定プロンプト	（後で入力）	TRUE	4	最高級鑑定30000円・溺愛体質変換（10000文字以上）

====================================
■ 基本情報（A6〜H6に貼り付け）
====================================

prompt_subscription	love_reading	SUBSCRIPTION_MONTHLY	溺愛サポート月次配信プロンプト	（後で入力）	TRUE	5	サブスク月次配信（1500-2000文字）

====================================
次のステップ：E列にプロンプトを貼り付け
====================================

1. E2セルをクリック
2. prompt_FREE_001.txt を開く
3. 全文をコピー
4. E2セルに貼り付け（Ctrl+V / Cmd+V）
5. Enterキー

同様に：
- E3セル ← prompt_PAID_5000.txt
- E4セル ← prompt_PAID_10000.txt
- E5セル ← prompt_PAID_30000.txt
- E6セル ← prompt_SUBSCRIPTION_MONTHLY.txt

完了！
