## 📱 Step 4-1: LINE公式アカウント作成（5分）

### 1. アカウント作成

1. [LINE for Business](https://www.linebiz.com/jp/) にアクセス
2. 「LINE公式アカウントを始める」をクリック
3. LINEでログイン
4. 「アカウントを作成」
5. 情報入力：
   ```
   アカウント名: いずみきょうか占い
   業種: 占い・スピリチュアル
   ```

### 2. 基本設定

LINE Official Account Manager で：

```
応答設定:
- 応答メッセージ: オフ
- あいさつメッセージ: オン
- Webhook: オン

プロフィール:
- ステータスメッセージ: あなたの運命を照らします✨
- プロフィール画像: アップロード
```

### 3. QRコード取得

「ホーム」→「友だち追加」→ QRコードをダウンロード

✅ LINE公式アカウント完成！

---

## 🔗 Step 4-2: プロラインフリー登録・連携（10分）

### 1. プロラインフリー登録

1. [プロラインフリー](https://www.pro-line.jp/) にアクセス
2. 「無料で始める」をクリック
3. メールアドレス登録
4. 認証メール→リンククリック
5. アカウント情報入力

### 2. LINE連携

1. プロライン管理画面にログイン
2. 「LINE連携」をクリック
3. 先ほど作成したLINE公式アカウントを選択
4. 「連携する」
5. 権限を許可

✅ プロライン連携完了！

## 🎨 Step 4-3: リッチメニュー作成（15分）

### リッチメニューの設計

```
┌─────────────────────────────┐
│    いずみきょうか占い        │
├────────────┬────────────────┤
│  無料鑑定   │   無料相談     │
│  🔮無料    │   💬Zoom 30分  │
├────────────┼────────────────┤
│有料鑑定（基本）│  有料鑑定（詳細）│
│ ⭐¥5,000  │  ✨¥10,000    │
└────────────┴────────────────┘
```

### プロラインで設定

1. プロライン管理画面 →「リッチメニュー」
2. 「新規作成」
3. テンプレート：「4分割」を選択
4. 画像をアップロード（Canva等で作成）
5. 各エリアにアクション設定：

#### エリア1（左上）：無料鑑定

```
アクションタイプ: リンク
URL: [L9002] 無料鑑定フォームのURL
```

#### エリア2（右上）：無料相談

```
アクションタイプ: リンク
URL: [L9001] 無料相談フォームのURL
```

#### エリア3（左下）：有料鑑定（基本）¥5,000

```
アクションタイプ: リンク
URL: https://paypal.me/izumikyouka/5000

※決済後のアクションで相談内容フォームへ誘導
```

#### エリア4（右下）：有料鑑定（詳細）¥10,000

```
アクションタイプ: リンク
URL: https://paypal.me/izumikyouka/10000

※決済後のアクションで相談内容フォームへ誘導
```

### 決済後のフォロー設定

プロラインで「シナリオ」→「カスタムシナリオ」を作成：

#### PayPal決済後のメッセージ（手動設定）

ユーザーが決済したら、運営が手動で以下のメッセージを送信：

```
{{name}}様

有料鑑定のお申し込み、
誠にありがとうございます！

お支払いを確認いたしました✨

次のステップとして、
ご相談内容を詳しくお聞かせください。

↓ こちらのフォームにご記入ください
{{form_url}}

※このフォームは1回のみ回答可能です
※じっくりお考えの上、ご記入ください

鑑定結果は7日以内にメールでお届けします。
楽しみにお待ちください🔮

いずみきょうか
```

**注意**: `{{form_url}}` は実際には、GASから送信されるメールに含まれます。

### より自動化したい場合（推奨フロー）

```
1. PayPal決済完了
2. ユーザーが「決済しました」とLINEメッセージ送信
3. プロラインの自動応答で案内：
   「お支払いありがとうございます！
    ご登録のメールアドレスに、相談内容記入フォームを
    お送りしておりますので、ご確認ください。」
```

この場合、実際の処理は：
```
PayPal決済完了
  ↓
運営がPayPal管理画面で確認
  ↓
manual_payments シートに記録
  ↓（1分以内に自動）
users/payments/applications に記録
  ↓
フォームURL案内メール自動送信
  ↓
ユーザーがメールからフォーム記入
```

✅ リッチメニュー完成！

## 📌 完成したシステム構成

### 全体フロー

```
【無料鑑定】
LINE → Googleフォーム → GAS → AI鑑定 → メール

【有料鑑定】
LINE → PayPal.Me → 決済 → manual_payments記録 → GAS → フォームURL送信 → ユーザーが記入 → AI鑑定 → メール

【無料相談】
LINE → Googleフォーム → GAS → 運営通知 → 日時確定 → Zoom面談
```

```
プロライン（フロント）
  ↓
Googleフォーム / PayPal
  ↓
GAS（処理エンジン）
  ↓
スプレッドシート（データベース）
  ↓
OpenAI API（鑑定生成）
  ↓
Gmail（配信）
  ↓
ユーザー
```

## 🆘 トラブルシューティング

### Q: リッチメニューが表示されない

**対処**:
1. プロラインで「公開」しているか確認
2. LINE公式アカウントをブロック→再度友だち追加
3. LINEアプリを再起動

### Q: PayPal.Meリンクが機能しない

**対処**:
1. PayPal.Meが有効になっているか確認
2. リンクの金額部分が正しいか確認
   ```
   OK: https://paypal.me/izumikyouka/5000
   NG: https://paypal.me/izumikyouka/5,000
   ```
3. PayPalアカウントが本人確認済みか確認

### Q: 決済後、自動処理されない

**対処**:
1. `manual_payments` シートに手動で記録
2. `processed` が FALSE になっているか確認
3. 1分待ってログを確認
4. `processManualPayments_()` が tick() に追加されているか確認

#### 毎日の作業（PayPal決済が入った場合）

   ```
   名前、メール、生年月日、商品名、金額、取引ID、日時、FALSE
   ```

3. 保存
4. 1分以内に自動処理される
5. ユーザーにメールが送信される
