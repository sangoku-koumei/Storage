## 🔄 毎日の作業（10分）

### 朝（5分）

```
1. PayPal管理画面を開く
2. 新しい決済を確認
3. manual_payments シートに記録：
   
   名前 | メール | 生年月日 | 商品名 | 金額 | 取引ID | 日時 | FALSE
   
4. 保存（1分以内に自動処理される）
```

### 昼or夜（5分）

```
1. send_queue シートでエラー確認
2. consult_requests シートで新規相談確認
3. 必要に応じて consult_decisions に記入
```

## 🆘 緊急時の対応

### システム全停止

```
config シートで:
freeze_all = TRUE

→ すべての自動処理が停止
```

### メール送信のみ停止

```
config シートで:
freeze_sending = TRUE

→ ユーザー宛メールは停止
→ 運営通知は継続
```

### アーカイブを手動実行

```
Apps Script エディタで:
runArchiveNow()

→ 即座にアーカイブ実行
```
