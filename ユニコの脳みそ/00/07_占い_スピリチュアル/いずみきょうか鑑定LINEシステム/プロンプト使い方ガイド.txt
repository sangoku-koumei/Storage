====================================
AIプロンプト 使い方ガイド
いずみきょうか鑑定LINEシステム
====================================

【ファイル構成】
プロンプトは商品ごとに分かれています。修正が簡単です！

📁 ai_prompts_管理シート.txt ← スプレッドシートに貼り付けるデータ
📄 prompt_FREE_001.txt ← 無料鑑定プロンプト
📄 prompt_PAID_5000.txt ← 5,000円鑑定プロンプト
📄 prompt_PAID_10000.txt ← 10,000円鑑定プロンプト
📄 prompt_PAID_30000.txt ← 30,000円鑑定プロンプト
📄 prompt_SUBSCRIPTION_MONTHLY.txt ← サブスク配信プロンプト

====================================
■ Step 1：スプレッドシートへの設定
====================================

1. Googleスプレッドシートを開く

2. 「ai_prompts」という名前のシートを作成

3. 「ai_prompts_管理シート.txt」を開く

4. 【ここからコピー】から【ここまでコピー】の部分をコピー

5. スプレッドシートのA1セルを選択して貼り付け（Ctrl+V / Cmd+V）

6. 正しく貼り付けられたか確認：
   ✓ A列：商品ID（FREE_001, PAID_5000...）
   ✓ B列：プロンプト名
   ✓ C列：プロンプトファイル名
   ✓ D列：使用目的
   ✓ E列：文字数目安

====================================
■ Step 2：鑑定文の生成方法
====================================

【例：5,000円の鑑定依頼が来た場合】

1. 「prompt_PAID_5000.txt」を開く

2. プロンプト全体をコピー

3. お客様情報のプレースホルダーを実際の情報に置き換える：
   {顧客名} → 山田花子
   {生年月日} → 1995年8月20日
   {性別} → 女性
   {彼の名前} → 田中太郎
   {恋愛状況} → 片思い
   {悩みの詳細} → 彼に告白したいけど勇気が出ない
   {質問事項} → 彼の気持ちと告白のタイミングを知りたい

4. ChatGPTまたはClaudeに貼り付けて実行

5. 生成された鑑定文を確認・必要に応じて修正

6. お客様にLINEまたはメールで納品

====================================
■ プレースホルダー一覧
====================================

【共通】
{顧客名} = お客様の名前
{生年月日} = お客様の生年月日（例：1990年5月15日）
{性別} = お客様の性別

【無料鑑定】
{選択肢} = A:愛し方について/金星 or B:愛され方について/火星

【有料鑑定共通】
{彼の名前} = 彼のお名前
{彼の生年月日} = 彼の生年月日（分かれば）
{恋愛状況} = 片思い/復縁/交際中など
{悩みの詳細} = 具体的な悩み
{質問事項} = 知りたいこと

【10,000円以上】
{出生時刻} = 出生時刻（例：14:30）
{理想の関係} = どうなりたいか

【30,000円】
{出生地} = 出生地（例：東京都）
{恋愛履歴} = これまでの恋愛パターン
{現在の状況} = 現在の状況（詳細）
{変わりたい理由} = 変わりたい理由
{願望} = 叶えたい願い

【サブスク】
{配信年月} = 配信する年月（例：2025年2月）

【共通（納品時）】
{鑑定日} = 鑑定を行った日
{祈祷日} = 祈祷を行った日（該当商品のみ）

====================================
■ AI使用のコツ
====================================

【推奨AI】
✅ Claude 3.5 Sonnet - 長文生成に強い、深い洞察
✅ GPT-4 - 高品質、文字数制限に注意

【高品質な鑑定文を生成するコツ】
1. お客様情報は詳しく入力する
2. プロンプト全体をそのまま使う（一部だけ使わない）
3. 生成後、いずみきょうか様の声を追加する
4. ホロスコープは別途作成ツールで作る
5. タロット/オラクルは実際に引くと説得力UP

【ホロスコープ作成ツール】
- Astro.com (https://www.astro.com/)
- 無料で詳細なホロスコープ図が作れる

【タロット/オラクル】
- 実際にカードを引く（推奨）
- またはAIに「この状況に最適なカード」を選んでもらう

====================================
■ プロンプトのカスタマイズ方法
====================================

【簡単な修正】
各プロンプトファイル（prompt_xxx.txt）を直接編集

【例：文字数を変えたい】
「文字数：2,000〜3,000文字」
　↓
「文字数：3,000〜4,000文字」

【例：鑑定スタイルを変えたい】
「## 鑑定スタイル」の部分を編集

【例：出力形式を変えたい】
「## 出力形式」の部分を編集

修正したらファイルを保存するだけ！
スプレッドシートの修正は不要です。

■ よくある質問（FAQ）
====================================

Q1. プロンプトを編集したらスプレッドシートも更新が必要？
A1. 不要です。プロンプトファイルを編集するだけでOK。

Q2. ホロスコープの天体が分からない場合は？
A2. Astro.comで生年月日を入力すれば自動計算されます。
    出生時刻が分からない場合は正午（12:00）で計算。

Q3. タロットカードを持っていない場合は？
A3. AIに「この状況に最適なタロットカード」を選んでもらえます。
    ただし、実際に引いた方がリアリティがあります。

Q4. 恋愛以外の相談が来たら？
A4. 基本は恋愛特化ですが、プロンプトを少し修正すれば対応可能。
    30,000円商品は恋愛以外も積極的に対応。

Q5. AIが生成した鑑定文をそのまま使っていい？
A5. 必ず確認して、いずみきょうか様の声を追加してください。
    特に「深層リーディング」部分は手書きで温かみを。

Q6. プロンプトを新しく追加したい場合は？
A6. 既存のプロンプトファイルをコピーして編集。
    スプレッドシートにも1行追加。

■ トラブルシューティング
====================================

【問題】スプレッドシートに貼り付けがうまくいかない
【解決】タブ区切り形式を確認。
       メモ帳で開いてからコピーし直す。

【問題】AIが長文を生成しきれない
【解決】Claude 3.5 Sonnetを使う（長文に強い）。
       または、セクションごとに分けて生成。

【問題】ホロスコープの解釈が分からない
【解決】AIに「太陽が○○座の人の恋愛傾向」と質問。
       基本的な解釈はAIが補完してくれます。

【問題】生成された鑑定文が硬い
【解決】プロンプトの「鑑定スタイル」セクションを編集。
       「もっと親しみやすく」などを追加。
