## 🆘 問題が発生したら

### 緊急停止

config シートで：

```
freeze_all を TRUE に設定

→ すべての自動処理が停止
```

落ち着いて、以下を確認：

1. logs_[今月] シートでエラー内容を確認
2. Apps Script の「実行数」でエラー詳細を確認
3. FAQ.md のトラブルシューティングを参照
4. 5_運用マニュアル.md のトラブルシューティングを参照
