## 📋 ステップ3: 設定を変更（10分）⭐最重要

### configシートを開く

```
1. スプレッドシートの「config」シート（タブ）を開く

2. 黄色のセルのみ変更してください
```

### 変更する項目（9箇所）

#### 1. sender_name（占い師名）

```
現在: いずみきょうか
変更: あなたの占い師名

例: 星読みマリア
例: タロット師ユキ
```

#### 2. sender_email（送信元メール）

```
現在: izumi.kyouka.uranai@gmail.com
変更: あなたのGmailアドレス

※ Gmailを推奨します
※ 新規作成でもOK
```

#### 3. ops_email（運営メール）

```
現在: izumi.kyouka.uranai@gmail.com
変更: 通知を受け取るメールアドレス

※ sender_emailと同じでもOK
```

#### 4. openai_api_key（OpenAI APIキー）

```
現在: sk-proj-サンプルキー
変更: あなたのOpenAI APIキー

取得方法:
1. https://platform.openai.com/api-keys
2. サインアップ（メールアドレスで）
3. 「Create new secret key」をクリック
4. キーをコピーして貼り付け

※ クレジットカード登録が必要
※ 初回$5の無料クレジットあり
```

#### 5. paypal_base_url（PayPal.Me URL）

```
現在: https://paypal.me/izumikyouka
変更: https://paypal.me/あなたのID

取得方法:
1. PayPalにログイン
2. プロフィール > PayPal.Me
3. あなた専用のURLを作成
4. コピーして貼り付け
```

#### 6. preview_mode（プレビューモード）

```
現在: TRUE
変更: まずはTRUEのまま（テスト用）

本番開始時に FALSE に変更
```

#### 7. freeze_all（全停止フラグ）

```
現在: FALSE
変更: FALSEのまま
```

#### 8. freeze_sending（送信停止フラグ）

```
現在: FALSE
変更: FALSEのまま
```

#### 9. preview_to（プレビュー送信先）

```
現在: test@example.com
変更: あなたのメールアドレス

※ preview_mode=TRUEの時、ここに送信されます
```

**✅ 設定完了！**

※同様に、`プロンプト集_完全版.md` の使い方に従って、他の項目も調整してください。

## 🆘 トラブルシューティング

### Q: setupSheets()でエラーが出る

```
A: 権限承認が必要です

対処:
1. 「権限を確認」をクリック
2. 「詳細」→「プロジェクト名に移動」
3. 「許可」をクリック
4. 再度実行
```

### Q: tick()でエラーが出る

```
A: OpenAI APIキーが正しいか確認

対処:
1. configシートでopenai_api_keyを確認
2. 正しいキーか確認
3. https://platform.openai.com/api-keys で確認
```

### Q: メールが送信されない

```
A: preview_modeを確認

対処:
1. configシートでpreview_mode = TRUE確認
2. TRUEの場合、preview_toのメールに[PREVIEW]で送信される
3. 本番開始時にFALSEに変更
```

### Q: フォーム送信が反映されない

```
A: フォームとスプシの連携を確認

対処:
1. フォームの「回答」タブを開く
2. スプレッドシートアイコンをクリック
3. 正しいスプシに接続されているか確認
```
