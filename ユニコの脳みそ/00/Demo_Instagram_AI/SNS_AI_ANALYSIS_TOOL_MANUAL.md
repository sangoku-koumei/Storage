# Instagram AI分析ツール 説明書

[[00_知識マップ|⬅️ 知識マップへ戻る]]

本ツールは、自社および競合のInstagramデータを収集・分析し、AI Factory 帝国・最高戦略参謀によるデータ駆動型の戦略立案を支援するプロフェッショナル・ツールです。

## 🌟 主な機能と使い方

### 1. データ管理アーカイブ（ホーム画面）
ホーム画面の最上部にある「📁 過去データ管理アーカイブ」で、これまでに収集・保存したデータを一括管理できます。
- **読込**: 過去のデータを復元し、分析やAIチャットで使用可能にします。
- **名前変更**: ファイル名を分かりやすい名前に変更できます。
- **削除**: 不要になったデータを整理できます。

### 2. データ収集と外部インポート
「📥 データ収集」タブでは、新しいデータの収集に加え、外部からの持ち込みも可能です。
- **外部CSVインポート**: お手持ちのInstagram投稿データ（CSV形式）をアップロードして、本ツールの強力なAI分析にかけることができます。
- **自動マージ**: 「自分の投稿」と「競合データ」は自動的に統合され、1つの強力なデータベースとして扱われます。

### 3. AI軍師（帝国・最高戦略参謀）
「🤖 AIチャット」タブでは、月額500万円クラスの知能を持つAI参謀と対話できます。
- **劇薬ショートカット**: 赤い「🧨 劇薬改善案を提示せよ」ボタンを押すと、全データをもとに「認知科学に基づいた外科手術レベルの改善案」が即座に出力されます。
- **具体的指示**: 抽象的なアドバイスではなく、「1枚目の画像に書くべき具体的なコピー」や「心理学に基づいたアルゴリズムハック」を提示します。

### 4. APIキーの管理とセキュリティ
- **永続化**: AIチャットの「🔑 APIキーの管理」メニューから、入力したキーを `.env` ファイルに保存できます。次回以降、入力の手間が不要になります。
- **セキュリティ**: APIキーが入ったファイルはGitにアップロードされないよう自動的に保護されています（`.gitignore` 設定済み）。

## 🏹 戦況ステータス表示
画面上部の「🏹 戦況ステータス」には、現在分析対象となっているデータの内容が常に表示されます。
- 左側：現在「読込中」のデータの内訳
- 右側：サイドバーで「選択中」のファイルの中身（重ならずに内容を確認できます）
