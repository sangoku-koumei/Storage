# 00フォルダ（知識戦略の保管庫）執筆運用ルール

本ドキュメントは、 0ディレクトリ配下のファイルを作成編集する際の**絶対的なルール**です。
.agent/rules/writing_style.md の基本ルールを継承しつつ、以下の**特例的厳格ルール**を最優先で適用します。

## 1. 知識の完全性と具体化（省略厳禁）
 0フォルダは、AIとユーザーの共通脳となる「知識の源泉」です。曖昧さを徹底的に排除します。

- **省略の完全禁止**: 貼り付けられた文章や指示された内容は、要約せず、**一言一句漏らさず**記録して下さい。
- **「なぜ」の記述（コンテキストの保存）**:
    - 単なる「事実」や「指示」だけでなく、「**なぜその結論に至ったのか**」「**どのような経緯でその話が出たのか**（思考プロセス、背景、感情）」までを含めて、細かく具体化して書き記して下さい。
    - 未来の自分が読み返した際、当時の文脈（Context）を完全に再現できるようにします。
- **網羅性**: 情報や知識として、話の一部でも欠くことがないようにして下さい。「些細なこと」と自己判断して削除してはいけません。

## 2. リンクタグの起点拡充（アグレッシブリンキング）
知識のネットワークを爆発的に広げるため、**リンクの「起点（アンカー）」そのものを能動的に作り出す**意識を持って下さい。

- **核心フレーズのタグ化リンク化**:
    - 文章中に「良い言葉」「核心的なフレーズ」「独自の概念」を見つけたら、即座に [[用語]] や #用語 としてリンク/タグ化して下さい。
    - 既存のリンク先が存在しなくても構いません。**まずリンクを作り、そこを起点に将来の知識の網（ウェブ）を広げる**という意思を持ってリンクを設置して下さい。
    - （例）単に「マーケティング」とするのではなく、「[[魂を震わせるマーケティング]]」や「[[3秒の美学]]」のように、概念が凝縮された言葉を切り出してリンクにして下さい。

## 3. Zettelkasten（ツェッテルカステン）ルールの遵守
膨大な知識を有機的に結合させるため、以下の原則を守ります。

- **アトミック性（Atomicity）**: 1つのトピックについて、完結した単位で記述して下さい。ただし、前述の通り内容は省略せず「深く」記述します。
- **相互参照**: 新しい知識を記録する際は、必ず**他の関連する知識（ファイル）へのリンク**を含めて下さい。孤立した知識を作ってはいけません。

## 4. 記述言語の日本語統一
 0フォルダ内の全ての情報は、未来のユーザー（日本人）が直感的に理解できるよう、言語を統一します。

- **完全日本語記述**: ファイル内の文章、思考プロセス、説明、コメントに至るまで、**すべて日本語**で記述して下さい。
- **英語出力の禁止**: 専門用語であっても、可能な限り日本語の補足を入れるか、日本語の文脈の中で使用して下さい。
