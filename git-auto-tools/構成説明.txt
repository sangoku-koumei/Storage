Git自動プッシュツール - ファイル構成

このフォルダには、CursorでGit操作を自動化するためのツールが含まれています。

【ファイル一覧】

■ Git自動プッシュツール
1. git-auto-push.ps1
   - メインのPowerShellスクリプト
   - Git操作（add, commit, push）を自動化
   - PowerShellネイティブ構文のみ使用（&&などは使用していません）

2. README.md
   - 使い方ガイド
   - クイックスタート手順
   - トラブルシューティング

3. keybindings.json.example
   - キーボードショートカット設定の例
   - Ctrl + Shift + G でGitプッシュを実行可能に

■ 文字化け対応ツール
4. fix-git-encoding.ps1
   - Gitの文字化けを防ぐ設定を自動適用
   - 日本語ファイル名やコミットメッセージの文字化けを防止

5. detect-encoding.ps1
   - ファイルの文字コード（エンコーディング）を検出
   - UTF-8、Shift_JIS、EUC-JPなどを自動判定

6. convert-to-utf8.ps1
   - ファイルをUTF-8（BOMなし）に変換
   - バックアップ機能付き

7. fix-corrupted-files.ps1
   - 文字化けファイルを一括検出・修復
   - Markdownファイルを自動スキャン

8. 文字化け対応ツール-使い方.md
   - 文字化け対応ツールの詳細な使い方ガイド
   - トラブルシューティング情報

【使用方法】

■ Git自動プッシュ
- Cursorで Ctrl + Shift + P → "Tasks: Run Task" → "Git: すべてをプッシュ（add + commit + push）" を選択
- 詳細は README.md を参照

■ 文字化け対応
- Git設定の修正: "文字化け対応: Git設定を修正" タスクを実行（初回のみ）
- 文字化けファイルの検出: "文字化け対応: 文字化けファイルを検出" タスクを実行
- 自動修復: "文字化け対応: 文字化けファイルを検出・自動修復" タスクを実行
- 詳細は 文字化け対応ツール-使い方.md を参照

【注意事項】
- すべてのスクリプトはPowerShellネイティブ構文で記述されています
- && や || などのbash固有の構文は使用していません
