- Kindle執筆ヘルパー を改善

### 今日の修正内容まとめ

**説明ドキュメントの追加**

- ツール全体の動き・ルールの意味・使い方を整理した。
- `Kindle執筆ヘルパー_ツール説明.md` を新規作成。

**整形ルール（custom_rules.json）の拡張・上書き**

- **改行関連**
  - `layout_settings.max_consecutive_newlines = 2` を明示し、「空行は最大2行まで」に統一。
  - `"\n{3,}" → "\n\n"`（正規表現）を追加し、3行以上連続する改行は必ず2行に圧縮されるように強化。
- **文末記号の後に改行を1行入れる**
  - 。 / ！ / ？ / 」 のあとを、それぞれ `。\n` / `！\n` / `？\n` / `」\n` に置換するルールを追加し、文末で必ず1行改行されるように変更。
- **引用符・記号の正規化**
  - すでに `" / “” → 「」` の変換がある前提で、
    - `‘…’ / ''…'' → 『…』` に変換するルールを追加。
  - `・・ → ..` に変換し、全角中点2つを半角ピリオド2つに統一。

**メニュー画面ランチャーの追加（ツール起動フローの変更）**

- `Kindle執筆ヘルパー_メニュー.py` を新規作成。
- このスクリプトを起動すると GUI メニューが表示され、
  - 「整形スタート」ボタンを押したときにだけ `Kindle執筆ヘルパー.exe` が実行されるように変更。
- `input` フォルダの存在やファイル有無を簡易チェックし、エラーや完了はダイアログで案内。
